pīyuṣavarṣaśrījayadevakavikṛtaḥ

candrālokaḥ

uccairasyati mandatāmarasatāṃ jāgratkalaṅkairavadhvaṃsaṃ hastayate ca yā sumanasām ullāsinī mānase 
dhṛṣṭodyanmadanāśanārcir amalā lokatrayīdarśikā sā netratritayīva khaṇḍaparaśor vāgdevatā dīvyatu //

haṃ ho cinmayacittacandramaṇayaḥ saṃvardhayadhvaṃ rasān re re svairiṇi nirvicārakavite māsmat prakāśībhava 
ullāsāya vicāravīcinicayālaṅkāravārām nidheś candrālokamayaṃ svayaṃ vitanute pīyūṣavarṣaḥ kṛtī //

yuktyāsvādyalasadrasaikavasatiḥ sāhityasārasvatakṣīrāmbhodhir agādhatām upadadhat sevyaḥ samāśrīyatām 
śrīr asmād upadeśakauśalamayaṃ pīyūṣam asmājjagaj jāgradbhāsurapadmakeśarayaśaḥśītāṃśur asmād budhāḥ //

taṃ pūrvācāryasūryoktijyotiḥ stomodgamaṃ stumaḥ 
yaṃ prastūya prakāśante madguṇās trasareṇavaḥ //

nāśaṅkanīyam eteṣāṃ matam etena dūṣyate 
kiṃ tu cakṣur mṛgākṣīṇāṃ kajjaleneva bhūṣyate //

pratibhaiva śrutābhyāsasahitā kavitāṃ prati 
hetur mṛdambusambaddhā bījamālā latāmiva //

nirdoṣā lakṣaṇavatī sarītir guṇabhūṣaṇā 
sālaṃkārarasānekavṛttir vāk kāvyanāmabhāk //

aṅgīkaroti yaḥ kāvyaṃ śabdārthāv analaṃkṛtī 
asau na manyate kasmād anuṣṇam analaṃ kṛtī //

vibhaktyutpattaye yogyaḥ śāstrīyaḥ śabda iṣyate 
rūḍhayaugikatanmiśraiḥ prabhedaiḥ sa punas tridhā //

avyaktayoganiryogayogābhāsais tridhādimaḥ 
te ca vṛkṣādibhūvādimaṇḍapādyā yathākramam //

śuddhatanmūlasaṃbhinnaprabhedair yaugikas tridhā 
te ca bhrāntisphuratkāntikaunteyādisvarūpiṇaḥ //

tanmiśro 'nyonyasāmānyaviśeṣaparivartanāt 
nīradhiḥ paṅkajaṃ saudhaḥ sāgaro bhūruhaḥ śaśī //

kṣīranīradhir ākāśapaṅkajaṃ tena siddhayati 
vibhaktyantaṃ padaṃ vākyaṃ tadvyūho 'rthasamāptitaḥ //

yuktārthatāṃ tāṃ ca vinā khaṇḍavākyaṃ sa iṣyate 
vākyaṃ ca khaṇḍavākyaṃ ca padam ekam api kvacit //

dhūmavattvād iti yathā devety āmantraṇaṃ yathā 
vākyāny ekārthaviśrāntāny āhur vākyakadambakam //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
anenāsāv ādyaḥ sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ //

syāc ceto viśatā yena sakṣatā ramaṇīyatā 
śabde 'rthe ca kṛtonmeṣaṃ doṣam udghoṣayanti tam //

bhavec chrutikaṭur varṇaḥ śravaṇodvejane paṭuḥ 
saṃvidrate vyākaraṇaviruddhaṃ cyutasaṃskṛti //

aprayuktaṃ daivatādiśabde puṃlliṅgatādikam 
asamarthaṃ tu hantyādeḥ prayogo gamanādiṣu //

sa hanti hanta kāntāre kāntaḥ kuṭilakuntalaḥ 
nihatārthaṃ lohitādau śoṇitādiprayogataḥ //

vyanakty anucitārthaṃ yat padam āhus tadeva tat 
iyam adbhutaśākhyagrakelikautukavānarī //

nirarthakaṃ tuhītyādi pūraṇaikaprayojanam 
arthe vidadhadityādau dadhadādyam avācakam //

dhatte nabhastalaṃ bhāsvān aruṇaṃ taruṇaiḥ karaiḥ 
ekākṣaraṃ vinā bhūbhrūkṣmādikaṃ khatalādivat //

aślīlaṃ trividhaṃ vrīḍājugupsāmaṅgalātmanā 
āhlādasādhanaṃ vāyuḥ kāntānāśe bhavet katham //

syād dvyartham iha sandigdhaṃ nadyāṃ yānti patatriṇaḥ 
syād apratītaṃ śāstraikagamyaṃ vītānumādivat //

śithilaṃ śayane lilye maccitaṃ te śaśiśriyi 
mastapiṣṭakaṭīloṣṭagallādi grāmyam ucyate //

neyārthaṃ lakṣaṇātyantaprasarād amanoharam 
himāṃśor hāradhikkārajāgare yāmikāḥ karāḥ //

kliṣṭam artho yadīyo 'rthaśreṇiniḥśreṇim ṛcchati 
haripriyāpitṛvadhūpravāhapratimaṃ vacaḥ //

avimṛṣṭavidheyāṃśaḥ samāsapihite vidhau 
viśanti viśikhaprāyāḥ kaṭākṣāḥ kāmināṃ hṛdi //

aparādhīna ityādi viruddhamatikṛn matam 
anyasaṅgatam uttuṅgahāraśobhipayodharau //

rasādyanucite varṇe pratikūlākṣaraṃ viduḥ 
na mām aṅgada jānāsi rāvaṇaṃ raṇadāruṇam //

yasminn upahato lupto visarga iha tat tathā 
kusandhiḥ paṭav āgaccha visandhir nṛpatī imau //

hatavṛttam anukto 'pi chandodoṣaś cakāsti cet 
viśālalocane! paśyā-mbaraṃ tārātaraṅgitam //

nyūnaṃ tvatkhaḍgasaṃbhūtayaśaḥpuṣpaṃ nabhastaṭam 
adhikaṃ bhavataḥ śatrūn daśaty asilatāphaṇī //

kathitaṃ punaruktā vāk śyāmābjaśyāmalocanā 
vikṛtaṃ dūravivṛtair aiyaruḥ kuñjarāḥ puram //

patatprakarṣaṃ hīnānuprāsāditve yathottaram 
gambhīrārambhadambholipāṇir eṣaḥ samāgataḥ //

samāptapunarāttaṃ syād eṣa pīyūṣabhājanam 
netrānandī tuṣārāṃśur udety ambudhibāndhavaḥ //

ardhāntarapadāpekṣi krīḍānṛtyeṣu sasmitam 
moghārambhaṃ stumaḥ śambhum ardharambhoruvigraham //

abhavanmatayogaḥ syān na ced abhimato 'nvayaḥ 
yena baddho 'mbudhir yasya rāmasyānucarā vayam //

dviṣāṃ sampadam ācchidya yaḥ śatrūn samapūrayat 
asthānasthasamāsaṃ na vidvajjanamanoramam //

mithaḥ pṛthagvākyapadaiḥ saṃkīrṇaṃ yat tad eva tat 
vaktreṇa bhrājate rātriḥ kāntā candreṇa rājate //

brahmāṇḍaṃ tvadyaśaḥpūragarbhitaṃ bhūmibhūṣaṇa 
ākarṇaya payaḥpūrṇasuvarṇakalaśāyate //

bhagnaprakramam ārabdhaśabdanirvāhahīnatā 
akramaḥ kṛṣṇa pūjyante tvām anabhyarcya devatāḥ //

amatārthāntaraṃ mukhye 'mukhyenārthe virodhakṛt 
tyaktahāram uraḥkṛtvā śokenāliṅgitāṅganā //

apuṣṭārtho viśeṣye cen na viśeṣo viśeṣaṇāt 
viśanti hṛdayaṃ kāntākaṭākṣāḥ khañjanatviṣaḥ //

kaṣṭaḥ spaṣṭāvabodhārtham akṣamo vācyasannibhaḥ 
vyāhataś cedvirodhaḥ syān mithaḥ pūrvāparārthayoḥ //

sahasrapatramitraṃ te vaktraṃ kenopamīyate 
kutastatropamā yatra punaruktaḥ sudhākaraḥ //

duṣkramagrāmyasandigdhās trayo doṣāḥ kramād amī 
tvadbhaktaḥ kṛṣṇa! gaccheyaṃ narakaṃ svargam eva vā //

ekaṃ me cumbanaṃ dehi tava dāsyāmi kañcukam 
brūta kiṃ sevyatāṃ candramukhīcandrakirīṭayoḥ //

anaucityaṃ kīrtilatāṃ taraṅgayati yaḥ sadā 
prasiddhyā vidyayā vāpi viruddhaṃ dvividhaṃ matam //

nyasteyaṃ paśya kandarpapratāpadhavaladyutiḥ 
ketakī śekhare śambhor dhatte candrakalātulām //

sāmānyaparivṛttiḥ syāt kuṇḍalacchavivigrahā 
viśeṣaparivṛttiḥ syād vanitā mama cetasi //

tathā sahacarācāruviruddhānyonyasaṃgatī 
dhvāṅkṣāḥ santaś ca tanayaṃ svaṃ paraṃ ca na jānate //

sarojanetra putrasya mukhendum avalokaya 
pālayiṣyati te gotram asau narapurandaraḥ //

pade tadaṃśe vākyāṃśe vākye vākyakadambake 
yathānusāram abhyūhed doṣān śabdārthasaṃbhavān //

doṣam āpatitaṃ svānte prasarantaṃ viśṛṅkhalam 
nivārayati yas tredhā doṣāṅkuśam uśanti tam //

doṣe guṇatvaṃ tanute doṣatvaṃ vā nirasyati 
bhavantam athavā doṣaṃ nayaty atyājyatāmasau //

mukhaṃ candraśriyaṃ dhatte śvetaśmaśrukarāṅkuraiḥ 
atra hāsyarasoddeśe grāmyatvaṃ guṇatāṃ gatam //

tava dugdhābdhisaṃbhūteḥ kathaṃ jātā kalāṅkitā 
kavīnāṃ samayād vidyāviruddho 'doṣatāṃ gataḥ //

dadhāra gaurī hṛdaye devaṃ himakarāṅkitam 
atra śleṣodayān naiva tyājyaṃ hīti nirarthakam //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
dvitīyas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ //

alpākṣarā vicitrārthakhyātir akṣarasaṃhatiḥ 
uṣākāntenānugataḥ śūraḥ śaurir ayaṃ punaḥ //

śobhā khyātāpi yaddoṣo guṇakīrtyā niṣidhyate 
mudhā nindanti saṃsāraṃ kaṃsārir yatra pūjyate //

abhimāno vicāraś ced ūhitārthaniṣedhakṛt 
indur yadi kathaṃ tīvraḥ sūryo yadi kathaṃ niśi //

hetus tyaktvā bahūn pakṣān yuktyaikasyāvadhāraṇam 
nendur nārko 'yam aurvāgniḥ sāgarād utthito dahan //

pratiṣedhaḥ prasiddhānāṃ kāraṇānām anādaraḥ 
na yuddhena bhruvoḥ spande-naiva vīrā nipātitāḥ //

niruktaṃ syān nirvacanaṃ nāmnaḥ satyaṃ tathānṛtam 
īdṛśaiś caritai rājan satyaṃ doṣākaro bhavān //

syān mithyādhyavasāyaś ced asatī sādhyasādhane 
candrāṃśusūtragrathitāṃ nabhaḥpuṣpasrajaṃ vaha //

siddhiḥ khyāteṣu cen nāma kīrtyate tulyatoktaye 
yuvām eveha vikhyātau tvaṃ balair jaladhir jalaiḥ //

yuktir viśeṣasiddhiś ced vicitrārthāntarānvayāt 
navas tvaṃ nīradaḥ ko 'pi svarṇair varṣasi yan muhuḥ //

kāryaṃ phalopalambhaś ced vyāpārād vastuto 'tha vā 
asāv udeti śītāṃśur mānacchedāya subhruvām //

ityādi lakṣaṇaṃ bhūri kāvyasyāhur maharṣayaḥ 
svarṇabhrājiṣṇubhālatvaprabhṛtīva mahībhujaḥ //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
tṛtīyas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ //

śleṣo vighaṭamānārthaghaṭamānatvavarṇanam 
sa tu śābdaḥ sajātīyaiḥ śabdair bandhaḥ sukhāvahaḥ //

ullasattanutāṃ nīte 'nante pulakakaṇṭakaiḥ 
bhītayā mānavatyaiva śriyāśliṣṭaṃ hariṃ stumaḥ //

yasmād antaḥsthitaḥ sarvaḥ svayam artho 'vabhāsate 
salilasyeva sūktasya sa prasāda iti smṛtaḥ //

samatālpasamāsatvaṃ varṇādyais tulyatātha vā 
śyāmalā komalā bālā ramaṇaṃ śaraṇaṃ gatā //

samādhir arthamahimā lasadghanarasātmanā 
syād antarviśatā yena gātramaṅkuritaṃ satām //

mādhuryaṃ punaruktasya vaicitryaṃ cārutāvaham 
vayasya paśya paśyāsyāś cañcalaṃ locanāñcalam //

ojaḥ syāt prauḍhir arthasya saṃkṣepo vātibhūyasaḥ 
ripuṃ hatvā yaśaḥ kṛtvā tvadasiḥ kośam āviśat //

saukumāryam apāruṣyaṃ paryāyaparivartanāt 
sa kathāśeṣatāṃ yātaḥ samāliṅgya marutsakham //

udāratā tu vaidagdhyam agrāmyatvāt pṛthaṅmatā 
mānaṃ muñca priye kiṃcil locanāntam udañcaya //

śṛṅgāre ca prasāde ca kāntyarthavyaktisaṃgrahaḥ 
amī daśa guṇāḥ kāvye puṃsi śauryādayo yathā //

tilakādyam iva strīṇāṃ vidagdhahṛdayaṅgamam 
vyatiriktam alaṅkāraṃ prakṛter bhūṣaṇaṃ girām //

vicitralakṣaṇo nyāso nirvāhaḥ prauḍhir aucitī 
śāstrāntararahasyoktiḥ saṃgraho dik pradarśitā //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
caturthas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ //

śabdārthayoḥ prasiddhyā vā kaveḥ prauḍhivaśena vā 
hārādivad alaṅkāraḥ sanniveśo manoharaḥ //

svaravyañjanasaṃdohavyūhāmandohadohadā 
gaur jagajjāgradutsekā chekānuprāsabhāsurā //

āvṛttavarṇasampūrṇaṃ vṛttyanuprāsavad vacaḥ 
amandānandasandohasvacchandāspadamandiram //

lāṭānuprāsabhūr bhinnā-bhiprāyā punaruktatā 
yatra syān na punaḥ śatror garjitaṃ taj jitaṃ jitam //

ślokasyārdhe tadardhe vā varṇāvṛttir yadi dhruvā 
tadā matā matimatāṃ sphuṭānuprāsatā satām //

upameyopamānādāv arthānuprāsa iṣyate 
candanaṃ khalu govindacaraṇadvandvavandanam //

punaruktapratīkāśaṃ punaruktārthasannibham 
aṃśukāntaṃ śaśī kurvann ambarāntam upaity asau //

āvṛttavarṇastabakaṃ stavakandāṅkuraṃ kaveḥ 
yamakaṃ prathamā dhuryamādhuryavacaso viduḥ //

kāvyavitpravaraiścitraṃ khaḍgabandhādi lakṣyate 
teṣv ādyam ucyate ślokadvayīsajjanarañjikā //

kāminīva bhavet khaḍgalekhā cārukarālikā 
kāśmīrasekā raktāṅgī śatrukaṇṭhāntikāśritā //

upamā yatra sādṛśyalakṣmīr ullasati dvayoḥ 
hṛdaye khelator uccais tanvaṅgīstanayoriva //

upamānopameyatve yatraikasyaiva jāgṛtaḥ 
indur indurivetyādau bhaved evam ananvayaḥ //

paryāyeṇa dvayos tac ced upameyopamā matā 
dharmo 'rtha iva pūrṇaśrīr artho dharma iva tvayi //

vikhyātasyopamānasya yatra syād upameyatā 
indur mukham ivetyādau syāt pratīpopamā tadā //

upamāne tu līlādipadāḍhye lalitopamā 
tvannetrayugalaṃ dhatte līlāṃ nīlāmbujanmanoḥ //

anekasyārthayugmasya sādṛśyaṃ stabakopamā 
śrito 'smi caraṇau viṣṇor bhṛṅgas tāmarasaṃ yathā //

syāt saṃpūrṇopamā yatra dvayor api vidheyatā 
padmānīva vinidrāṇi netrāṇy āsann aharmukhe //

yatropamānacitreṇa sarvathāpy uparajyate 
upameyamayī bhittis tatra rūpakam iṣyate //

samānadharmayuksādhyāropāt sopādhirūpakam 
utsiktakṣitibhṛllakṣyapakṣacchedapurandaraḥ //

pṛthakkathitasādṛśyaṃ dṛśyaṃ sādṛśyarūpakam 
ullasatpañcaśākhas te rājate bhujabhūruhaḥ //

syād aṅgayaṣṭirityevaṃvidham ābhāsarūpakam 
aṅgayaṣṭidhanurvallītyādi rūpitarūpakam //

pariṇāmo 'nayor yasminn abhedaḥ paryavasyati 
kāntena pṛṣṭā rahasi maunam evottaraṃ dadau //

bahubhirbahudhollekhād ekasyollekhitā matā 
strībhiḥ kāmaḥ priyaiś candraḥ kālaḥ śatrubhir aikṣi saḥ //

atathyam āropayituṃ tathyāpāstir apahnutiḥ 
nāyaṃ sudhāṃśuḥ kiṃ tarhi vyomagaṅgāsaroruham //

paryastāpahnutir yatra dharmamātraṃ niṣidhyate 
nāyaṃ sudhāṃśuḥ kiṃ tarhi sudhāṃśuḥ preyasīmukham //

bhrāntāpahnutir anyasya śaṅkayā tathyanirṇaye 
tāpaṃ tanoti sotkampaṃ jvaraḥ kiṃ na sakhi smaraḥ //

chekāpahnutir anyasya śaṅkayā tathyanihnave 
prajalpan matpade lagnaḥ kāntaḥ kiṃ na hi nūpuraḥ //

kaitavāpahnutir vyakte vyājādyair nihnave padaiḥ 
niryānti smaranārācāḥ kāntādṛkpātakaitavāt //

utprekṣonnīyate yatra hetvādir nihnutiṃ vinā 
tvanmukhaśrīkṛte nūnaṃ padmair vairāyate śaśī //

ivādikapadābhāve gūḍhotprekṣāṃ pracakṣate 
yatkīrtir vibhramaśrāntā viveśa svarganimnagām //

syāt smṛtibhrāntisaṃdehais tad evālaṃkṛtitrayam 
paṅkajaṃ paśyatas tasyā mukhaṃ me gāhate manaḥ //

ayaṃ pramattamadhupas tvanmukhaṃ veda paṅkajam 
paṅkajaṃ vā sudhāṃśur vety asmākaṃ tu na nirṇayaḥ //

mīlitaṃ bahusādṛśyād bhedavac cen na lakṣyate 
raso nālakṣi lākṣāyāś caraṇe sahajāruṇe //

sāmānyaṃ yadi sādṛśyād bheda eva na lakṣyate 
padmākarapraviṣṭānāṃ mukhaṃ nālakṣi subhruvām //

hetoḥ kuto 'pi vaiśiṣṭyāt sphūrtir unmīlitaṃ matam 
lakṣitāny udite candre padmāni ca mukhāni ca //

anumānaṃ ca kāryādeḥ kāraṇādyavadhāraṇam 
asti kiñcid yadanayā māṃ vilokya smitaṃ manāk //

arthāpattiḥ svayaṃ sidhyet padārthāntaravarṇanam 
sa jitas tvanmukhenenduḥ kā vārtā sarasīruhām //

syāt kāvyaliṅgaṃ vāgartho nūtanārthasamarthakaḥ 
jito 'si manda kandarpa maccitte 'sti trilocanaḥ //

alaṅkāraḥ parikaraḥ sābhiprāye viśeṣaṇe 
sudhāṃśukalitottaṃsas tāpaṃ haratu vaḥ śivaḥ //

sābhiprāye viśeṣye tu bhavet parikarāṅkuraḥ 
caturṇāṃ puruṣārthānāṃ dātā devaś caturbhujaḥ //

akramātiśayoktiś ced yugapat kāryakāraṇe 
āliṅganti samaṃ deva jyāṃ śarāś ca parāś ca te //

atyantātiśayoktis tatpaurvāparyavyatikrame 
agre māno gataḥ paścād anunītā priyeṇa sā //

capalātiśayoktis tu kārye hetuprasaktije 
yāmīti priyapṛṣṭāyā valayo 'bhavad ūrmikā //

sambandhātiśayoktiḥ syāt tadabhāve 'pi tadvacaḥ 
paśya saudhāgrasaṃsaktaṃ vibhāti vidhumaṇḍalam //

bhedakātiśayoktiś ced ekasyaivānyatocyate 
aho anyaiva lāvaṇyalīlā bālākucasthale //

rūpakātiśayoktiś ced rūpyaṃ rūpakamadhyagam 
paśya nīlotpaladvandvān niḥsaranti śitāḥ śarāḥ //

prauḍhoktis tadaśaktasya tacchaktatvāvakalpanam 
kalindajātīraruhāḥ śyāmalāḥ saraladrumāḥ //

sambhāvanaṃ yadītthaṃ syād ity ūhonyaprasiddhaye 
siktaṃ sphaṭikakumbhāntaḥsthitiśvetīkṛtair jalaiḥ 
mauktikaṃ cel latāṃ sūte tatpuṣpais te samaṃ yaśaḥ //

vāñchitād adhikaprāptir ayatnena praharṣaṇam 
dīpam uddyotayed yāvat tāvad abhyudito raviḥ //

iṣyamāṇaviruddhārthasamprāptis tu viṣādanam 
dīpam uddyotayed yāvat tāvan nirvāṇa eva saḥ //

kriyādibhir anekasya tulyatā tulyayogitā 
saṅkucanti sarojāni svairiṇīvadanāni ca 
prācīnācalacūḍāgracumbibimbe sudhākare //

prastutāprastutānāṃ ca tulyatve dīpakaṃ matam 
medhāṃ budhaḥ sudhām indur bibharti vasudhāṃ bhavān //

āvṛtte dīpakapade bhaved āvṛttidīpakam 
dīptyāgnir bhāti bhātīnduḥ kāntyā bhāti ravis tviṣā //

vākyayor arthasāmānye prativastūpamā matā 
tāpena bhrājate sūraḥ śūraś cāpena rājate //

ced bimbapratibimbatvaṃ dṛṣṭāntas tadalaṃkṛtiḥ 
syān mallapratimallatve saṅgrāmoddhāmahuṃkṛtiḥ //

dṛṣṭāntaś ced bhavanmūrtis tan mṛṣṭā daivadurlipiḥ 
jātā cet prāk prabhā bhānos tarhi yātā vibhāvarī //

vākyārthayoḥ sadṛśayor aikyāropo nidarśanā 
yā dātuḥ saumyatā seyaṃ sudhāṃśor akalaṅkatā //

vyatireko viśeṣaś ced upamānopameyayoḥ 
śailā ivonnatāḥ santaḥ kiṃ tu prakṛtikomalāḥ //

sahoktiḥ sahabhāvaś ced bhāsate janarañjanaḥ 
digantam agamad yasya kīrtiḥ pratyarthibhiḥ saha //

vinoktiś ced vinā kiṃcit prastutaṃ hīnam ucyate 
vidyā hṛdyāpi sāvadyā binā vinayasampadam //

samāsoktiḥ parisphūrtiḥ prastute 'prastutasya cet 
ayam aindrīmukhaṃ paśya raktaś cumbati candramāḥ //

khaṇḍaśleṣaḥ padānāṃ ced ekaikaṃ pṛthagarthatā 
ucchaladbhūrikīlālaḥ śuśubhe vāhinīpatiḥ //

bhaṅgaśleṣaḥ padastomasy-aiva cet pṛthagarthatā 
ajarāmaratā kasya nāyodhyeva purī priyā //

arthaśleṣo 'rthamātrasya yady anekārthasaṃśrayaḥ 
kuṭilāḥ śyāmalā dīrghā kaṭākṣāḥ kuntalāś ca te //

aprastutapraśaṃsā syāt sā yatra prastutānugā 
kāryakāraṇasāmānyaviśeṣāder asau matā //

kamalaiḥ kamalāvāsaiḥ kiṃ kiṃ nāsādi sundaram 
apy ambudheḥ paraṃ pāraṃ prayānti vyavasāyinaḥ //

bhaved arthāntaranyaso 'nuṣaktārthāntarābhidhā 
hanūmān abdhim atarad duṣkaraṃ kiṃ mahātmanām //

yasmin viśeṣasāmānyaviśeṣāḥ sa vikasvaraḥ 
sa na jigye mahānto hi durdharṣāḥ kṣmādharā iva //

kāryādyaiḥ prastutair uktaiḥ paryāyoktiṃ pracakṣate 
tṛṇāny aṅkurayāmāsa vipakṣanṛpasadmasu //

uktir vyājastutir nindāstutibhyāṃ stutinindayoḥ 
kaste viveko nayasi svargaṃ pātakino 'pi yat //

ākṣepas tu prayuktasya pratiṣedho vicāraṇāt 
candra saṃdarśayātmānam athavāsti priyāmukham //

gūḍhākṣepo vidhau vyakte niṣedhe cāsphuṭe sati 
hara sītāṃ sukhaṃ, kiṃ tu cintayāntakaḍhaukanam //

virodho 'nupapattiś ced guṇadravyakriyādiṣu 
amandacandanasyandaḥ svacchandaṃ dandahīti mām //

śleṣādibhūr virodhaś ced virodhābhāsatā matā 
apy andhakāriṇānena jagad etat prakāśate //

asaṃbhavo 'rthaniṣpattāv asaṃbhāvyatvavarṇanam 
ko veda gopaśiśukaḥ śailam utpāṭayiṣyati //

vibhāvanā vināpi syāt kāraṇaṃ kāryajanma cet 
paśya lākṣārasāsiktaṃ raktaṃ tvaccaraṇadvayam //

viśeṣoktir anutpattiḥ kāryasya sati kāraṇe 
namantam api dhīmantaṃ na laṅghayati kaścana //

ākhyāte bhinnadeśatve kāryahetvor asaṃgatiḥ 
tvadbhaktānāṃ namaty aṅgaṃ bhaṅgam eti bhavaklamaḥ //

viṣamaṃ yady anaucityād anekānvayakalpanam 
kvātitīvraviṣāḥ sarpāḥ kvāsau candanabhūruhaḥ //

samam aucityato 'nekavastusambandhavarṇanam 
anurūpaṃ kṛtaṃ sadma hāreṇa kucamaṇḍalam //

vicitraṃ cet prayatnaḥ syād viparītaphalapradaḥ 
namanti santas trailokyād api labdhuṃ samunnatim //

adhikaṃ bodhyam ādhārād ādheyādhikavarṇanam 
yayā vyāptaṃ jagat tasyāṃ vāci mānti na te guṇāḥ //

anyonyaṃ nāma yatra syād upakāraḥ parasparam 
triyāmā śaśinā bhāti śaśī bhāti triyāmayā //

viśeṣaḥ khyātam ādhāraṃ vināpy ādheyavarṇanam 
gate 'pi sūrye dīpasthās tamaś chindanti tatkarāḥ //

syād vyāghāto 'nyathākāri vastv anyakriyam ucyate 
yair jagat prīyate hanti tair eva kusumāyudhaḥ //

gumphaḥ kāraṇamālā syād yathāprākprāntakāraṇaiḥ 
nayena śrīḥ śriyā tyāgas tyāgena vipulaṃ yaśaḥ //

gṛhītamuktarītyarthaśreṇir ekāvalī matā 
netre karṇāntaviśrānte karṇau dormūladolinau //

dīpakaikāvalīyogān mālādīpakam ucyate 
smareṇa hṛdaye tasyās tena tvayi kṛtā sthitiḥ //

sāro nāma padotkarṣaḥ sāratāyā yathottaram 
sāraṃ sārasvataṃ tatra kāvyaṃ tatra śivastavaḥ //

udārasāraś ced bhāti bhinno 'bhinnatayā guṇaḥ 
madhuraṃ madhu pīyūṣaṃ tasmāt tasmāt kaver vacaḥ //

yathāsaṃkhyaṃ dvidhārthāś cet kramādekaikamanvitāḥ 
śatruṃ mitraṃ dviṣatpakṣaṃ jaya rañjaya bhañjaya //

paryāyaś ced anekatra syād ekasya samanvayaḥ 
padmaṃ muktvā gatā candraṃ kāminīvadanopamā //

parivṛttir vinimayo nyūnābhyadhikayor mithaḥ 
jagrāhaikaṃ śaraṃ muktvā kaṭākṣān śatruyoṣitām //

parisaṃkhyā niṣidhyaikam anyasmin vastuyantraṇam 
snehakṣayaḥ pradīpeṣu svānteṣu na natabhruvām //

vikalpas tulyabalayor virodhaś cāturīyutaḥ 
kāntācitte 'dhare vāpi kuru tvaṃ vītarāgatām //

bhūyasām ekasambandhabhājāṃ gumphaḥ samuccayaḥ 
naśyanti paścāt paśyanti bhraśyanti ca tava dviṣaḥ //

samādhiḥ kāryasaukaryaṃ kāraṇāntarasannidheḥ 
utkaṇṭhitāṃ ca kalayan jagāmāstaṃ ca bhānumān //

pratyanīkaṃ balavataḥ śatroḥ pakṣe parākramaḥ 
jaitranetrānugau karṇāv utpalābhyām adhaḥkṛtau //

pratīpam upamānasya hīnatvam upameyataḥ 
dṛṣṭaṃ ced vadanaṃ tasyāḥ kiṃ padmena kimindunā //

ullāso 'nyamahimnā ced doṣo hy anyatra varṇyate 
tad abhāgyaṃ dhanasyaiva yan nāśrayati sajjanam //

tadguṇaḥ svaguṇatyāgād anyataḥ svaguṇodayaḥ 
padmarāgāruṇaṃ nāsāmauktikaṃ te 'dharaśritam //

punaḥ svaguṇasamprāptir vijñeyā pūrvarūpatā 
harakaṇṭhāṃśulipto 'pi śeṣas tvadyaśasā sitaḥ //

yadvastuno 'nyathā rūpaṃ tathā syāt pūrvarūpatā 
dīpe nirvāpite hy āsīt kāñcīratnair aharmahaḥ //

saṅgatānyaguṇānaṅgīkāram āhur atadguṇam 
viśann api raver madhyaṃ śīta eva sadā śaśī //

prāksiddhasvaguṇotkarṣo 'nuguṇaḥ parasannidheḥ 
karṇotpalāni dadhate kaṭākṣairapi nīlatām //

avajñā varṇyate vastu guṇadoṣākṣamaṃ yadi 
mlāyanti yadi padmāni kā hānir amṛtadyuteḥ //

praśnottaraṃ krameṇoktau syūtam uttaram uttaram 
yatrāsau vetasī pāntha tatrāsau sutarā sarit //

pihitaṃ paravṛttāntajñātur anyasya ceṣṭitam 
priye gṛhāgate prātaḥ kāntā talpam akalpayat //

vyājoktiḥ śaṅkamānasya chadmanā vastugopanam 
sakhi paśya gṛhārāmaparāgair asmi dhūsarā //

vakroktiḥ śleṣakākubhyāṃ vācyārthāntarakalpanam 
muñca mānaṃ dinaṃ prāptaṃ manda nandī harāntike //

svābhāvoktiḥ svabhāvasya jātyādiṣu ca varṇanam 
kuraṅgair uttaraṅgākṣi stabdhakarṇair udīkṣyate //

bhāvikaṃ bhūtabhāvyarthasākṣād darśanavarṇanam 
alaṃ vilokayādyāpi yuddhyante 'tra surāsurāḥ //

deśātmaviprakṛṣṭasya darśanaṃ bhāvikacchaviḥ 
tvaṃ vasan hṛdaye tasyāḥ sākṣāt pañceṣur īkṣyase //

udāttam ṛddheś caritaṃ ślāghyaṃ cānyopalakṣaṇam 
sānau yasyābhavad yuddhaṃ tad dhūrjaṭikirīṭinoḥ //

atyuktir adbhutātathyaśauryaudāryādivarṇanam 
tvayi dātari rājendra yācakāḥ kalpaśākhinaḥ //

rasabhāvatadābhāsabhāvaśāntinibandhanāḥ 
rasavatpreyaūrjasvisamāhitamayābhidhāḥ //

bhāvānāmudayaḥ sandhiḥ śabalatvam iti trayaḥ 
alaṅkārān imān sapta kecid āhur manīṣiṇaḥ //

śuddhir ekapradhānatvaṃ tathā saṃsṛṣṭisaṃkarau 
eteṣām eva vinyāsān nālaṅkārāntarāṇy amī //

sarveṣāṃ ca pratidvandvapraticchandabhidābhṛtām 
upādhiḥ kvacid udbhinnaḥ syād anyatrāpi saṃbhavāt //

mālā paraṃparā caiṣāṃ bhūyasām anukūlake 
manuṣye bhavataḥ kvāpi hy alaṅkārāṅgatāṃ gate //

śabde padārthe vākyārthe vākyārthastabake tathā 
ete bhavanti vinyāsāḥ svabhāvātiśayātmakāḥ //

kasyāpy atiśayasyokter ity anvarthavicāraṇāt 
prāyeṇāmī hy alaṃkārā bhinnā nātiśayoktitaḥ //

alaṃkārapradhāneṣu dadhāneṣv api sāmyatām 
vailakṣaṇyaṃ prativyakti pratibhāti mukheṣviva //

alaṃkāreṣu tathyeṣu yady anāsthā manīṣiṇām 
tad arvācīnabhedeṣu nāmnāṃ nāmnāya iṣyatām //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
caturthaḥ saikoyam sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ //

ālambanoddīpanātmā vibhāvaḥ kāraṇaṃ dvidhā 
kāryo 'nubhāvo bhāvaś ca sahāyo vyabhicāry api //

galadvedyāntarodbhedo hṛdayeṣv ajaḍātmanām 
milan malayajālepa ivāhlādaṃ vikāsayan //

kāvye nāṭye ca kārye ca vibhāvādyair vibhāvitaḥ 
āsvādyamānaikatanuḥ sthāyī bhāvo rasaḥ smṛtaḥ //

ratyākhyasthāyibhāvātmā vallabhādivibhāvitaḥ 
ālasyerṣyājugupsābhyo vinā saṃcāribhir yutaḥ //

anubhāvaiḥ kaṭākṣādyair unmādādyair yathākramam 
sambhogo vipralambhaś ca śṛṅgāro dvividho mataḥ //

hāsasthāyī raso hāsyo vibhāvadyair yathākramam 
vairūpyaphullagaṇḍatvā-vahitthādyaiḥ samanvitaḥ //

abhīṣṭaviprayogāśrupātaglānyādibhiḥ kramāt 
vibhāvādyair yutaḥ śokasthāyī syāt karuṇo rasaḥ //

krodhasthāyī raso raudro vibhāvādyaiḥ samanvitaḥ 
mātsaryahastaniṣpeṣasaṃmohādyair yathākramam //

utsāhākhyasthāyibhāvaḥ prabhāvādivibhāvabhūḥ 
vīro 'nubhāvaiḥ sthairyādyair bhāvair garvādibhiryutaḥ //

vyāghrādibhir vibhāvais tu vepitādyanubhāvabhṛt 
bhāvair mohādibhir yukto bhayasthāyī bhayānakaḥ //

sthāyī jugupsā bībhatso vibhāvādyā yathākramam 
aniṣṭekṣaṇaniṣṭhīvamohādyā yatra saṃmatāḥ //

adbhuto vismayasthāyī māyādikavibhāvabhūḥ 
romāñcādyanubhāvo 'yaṃ stambhādivyabhicārikaḥ //

nirvedasthāyikaḥ śāntaḥ satsaṅgādivibhāvabhūḥ 
kṣamādikānubhāvo 'yaṃ stambhādivyabhicārikaḥ //

ratir devādiviṣayā santi ca vyabhicāriṇaḥ 
vedyamānā nigadyante bhāvāḥ sāhityavedibhiḥ //

nirvedaglāniśaṅkākhyās tathāsūyāmadaśramāḥ 
ālasyaṃ caiva dainyaṃ ca cintā mohaḥ smṛtir dhṛtiḥ //

vrīḍā capalatā harṣa āvego jaḍatā tathā 
garvo viṣāda autsukyaṃ nidrāpasmāra eva ca //

suptaṃ prabodho 'marṣaś cāpy avahittham athogratā 
matir vyādhis tathonmādas tathā maraṇam eva ca //

trāsaś caiva vitarkaś ca vijñeyā vyabhicāriṇaḥ 
trayastriṃśad ime bhāvāḥ samākhyātās tu nāmataḥ //

sarvasādhāraṇapremapraśrayādisvarūpayā 
anaucityā rasābhāsā bhāvābhāsāś ca kīrtitāḥ //

bhāvasya śāntir udayaḥ sandhiḥ śabalatā tathā 
kāvyasya kāñcanasyeva kuṅkumaṃ kāntisaṃpade //

āturyam āsaptamaṃ ca yatheṣṭair aṣṭamādibhiḥ 
samāsaḥ syāt padair na syāt samāsaḥ sarvathāpi ca //

pāñcālikī ca lāṭīyā gauḍīyā ca yathārasam 
vaidarbhī ca yathāsaṃkhyaṃ catasro rītayaḥ smṛtāḥ //

madhurāyāṃ samākrāntā vargasthāḥ pañcamair nijaiḥ 
lakāraś ca lasaṃyukto hrasvavyavahitau raṇau //

rephākrāntā vargyayaṇāṣ ṭavargāt pañcamād ṛte 
kapākrāntas tavargaḥ syāt prauḍhāyāṃ ca kamūrdhatā //

sarvair ūrdhvaiḥ sakārasya sarvai rephasya sarvathā 
rahor dvedhā tu saṃyogaḥ paruṣāyāṃ śaṣau svataḥ //

lakāro 'nyair asaṃyukto laghavo ghabhadhā rasau 
lalitāyāṃ tathā śeṣā bhadrāyām iti vṛttayaḥ //

aṅgabhaṅgollasallīlā taruṇī smaratoraṇam 
tarkakarkaśapūrṇoktiprāptotkaṭadhiyāṃ vṛthā //

vīpsotsarpanmukhāgrārdraṃ barhī jahre kṛśas tṛṣam 
lalanā rabhasaṃ dhatte ghanāṭope mahīyasi //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mahati ṛtusaṃkhyaḥ //

vṛttibhedais tribhir yuktā srotobhiriva jāhnavī 
bhāratī bhāti gambhīrā kuṭilā saralā kvacit //

sāṃmukhyaṃ vidadhānāyāḥ sphuṭam arthāntare giraḥ 
kaṭākṣa iva lolākṣyā vyāpāro vyañjanātmakaḥ //

avivakṣitavācyasya dvau bhedau vācyam eva cet 
arthāntire saṃkramitam atyantaṃ vā tiraskṛtam //

dvau vivakṣitavācyasya lakṣyālakṣyakramātmakau 
catvāriṃśadyutaikena bhedāḥ ṣaṭ cānayoḥ kramāt //

tridhā śabdārthatadyugmaśaktijanmā sphuṭakramāt 
rasabhāvatadābhāsapramukhas tv asphuṭakramāt //

vastvalaṅkārayor vyakter bhedau dvau śabdaśaktijau 
arthaśaktisamutthasya bhedā dvādaśa tad yathā //

catvāro vastv alaṃkāram alaṃkāras tu vastu yat 
alaṃkāram alaṃkāro vastu vastu vyanakti tat //

vaktuḥ kavinibaddhasya kaver vā prauḍhinirmitaḥ 
svasiddho vā vyañjako 'rthaś catvāras triguṇās tataḥ //

śabdārthobhayabhūr ekaḥ sa ca vākyaikasaṃśrayaḥ 
padaikadeśe racanāvarṇavākyapadeṣv api //

prabandhe ceti ṣoḍhāsau rasādyākhyo 'sphuṭakramaḥ 
eṣu saptadaśasv ekaṃ parityajyāsphuṭakramam //

ye ṣoḍaśādyā dviguṇās te syur vākyapadāśrayāt 
prabandhe 'pi dvādaśa syur arthaśaktibhuvo bhidaḥ //

dvātriṃśad dvādaśaikaḥ ṣaṭ sarvasaṃkalitadhvaneḥ 
bhedāḥ syur ekapañcāśat saṃbhinnās tu sahasraśaḥ //

vaktṛsyūtaṃ bodhayituṃ vyaṅgyaṃ vaktur abhīpsitam 
svāṅkuritam atadrūpaṃ svayam ullasitaṃ giraḥ //

kaścit sādhāraṇaḥ kaścid āmantrya pratibodhitaḥ 
kaścit taṭasthaḥ kaścic ca bodhitapratibodhitaḥ //

ityevaṃ boddhṛvaicitryād vaktṛsyūtaṃ caturvidham 
upekṣānihnavābhyāṃ ca dvidhā svāṅkuritaṃ matam //

bhūtādikālabhedena nihnavaḥ syād anekadhā 
anenāpi prabhedena vyaktivallī vijṛmbhate //

nānāprabhedā niyatā kvacit prakaraṇādinā 
arthe 'rtham anyaṃ yaṃ vakti tad vācyavyaṅgyam iṣyate //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mahati munisaṃkhyaḥ //

yad vyajyamānaṃ manasaḥ staimityāya sa no dhvaniḥ 
anyathā tu guṇībhūtavyaṅgyam āpatitaṃ tridhā //

vyaktir eva kvacid vyaṅgyaḥ kvacid arthasvabhāvataḥ 
kvacic cārutarasyāgre sa vimuñcati cārutām //

agūḍhaṃ kalayed arthā-ntarasaṃkramitādikam 
vismṛtaḥ kim apāṃnātha sa tvayā kumbhasaṃbhavaḥ //

aparasya rasādeś ced aṅgamanyad rasādikam 
hā hā! matkucakāśmīraliptaṃ bhinnam uraḥ śaraiḥ //

tathā vācyasya siddhyaṅgaṃ naur artho vāridher yathā 
saṃśritya taraṇiṃ dhīrās taranti vyādhivāridhīn //

asphuṭaṃ stanayor atra kokasādṛśyavan matam 
kuṅkumāktaṃ stanadvandvaṃ mānasaṃ mama gāhate //

saṃdigdhaṃ yadi saṃdeho dairghyādyutpalayoriva 
saṃprāpte nayane tasyāḥ śravaṇottaṃsabhūmikām //

tulyaprādhānyam indutvam iva vācyena sāmyabhṛt 
kānte tvadānanarucā glānim eti saroruham //

asundaraṃ yadi vyaṅgyaṃ syād vācyād amanoharam 
sarasyāmīladambhoje cakraḥ kāntāṃ vilokate //

kākusthaṃ praṇato 'mbhodhir adya mādyatu rāvaṇaḥ 
ity aṣṭadhā guṇībhūtavyaṅgyam aṅgīkṛtaṃ budhaiḥ //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke mahati vasusaṃkhyaḥ sukhayatu //

mukhyārthasyāvivakṣāyāṃ pūrvārvācī ca rūḍhitaḥ 
prayojanāc ca saṃbaddhaṃ vadantī lakṣaṇā matā //

lakṣaṇīyasvaśabdasya mīlanāmīlanād dvidhā 
lakṣaṇā sā tridhā siddhasādhyasādhyāṅgabhedataḥ //

sphuṭāsphuṭaprabhedena prayojanam api dvidhā 
viduḥ sphuṭaṃ taṭasthatvād arthagatvād dvidhā budhāḥ //

asphuṭaṃ cārthaniṣṭhatvāt taṭasthatvād api dvidhā 
lakṣyalakṣakaniṣṭhatvād arthasaṃstham api dvidhā //

lakṣakasthaṃ sphuṭaṃ yatra sā vicakṣaṇalakṣaṇā 
asphuṭatvaṃ taṭasthatvaṃ lakṣyasthatvam amuṣya ca //

anyās tisra iti vyaktā śaktitaḥ sā caturvidhā 
indur evaiṣa tadvaktram utkarṣo lakṣyate mukhe //

pradīpaṃ vardhayetyatra taṭasthaṃ maṅgalodayaḥ 
paṭo 'yaṃ dagdha ityādau sphuṭaṃ nāsti prayojanam //

amṛtaṃ sūktam ityādau lakṣyastham atihṛdyatā 
ābhimukhyāt saṃnidhānāt tathākārapratītitaḥ //

kāryakāraṇabhāvāt sā vācyavācakabhāvataḥ 
ityevamādeḥ saṃbandhāt kiṃcānyasmāc catuṣṭayāt //

sādṛśyāt samavāyāt sā vaiparītyāt kriyānvayāt 
sāropādhyavasānākhye gauṇaśuddhe pṛthak pṛthak //

gauṇaṃ sāropam uddiṣṭam indur mukham itīdṛśam 
gauṇaṃ sādhyavasānaṃ syād indur evedam īdṛśam //

śuddhaṃ sāropam uddiṣṭam āyur ghṛtam itīdṛśam 
śuddhaṃ sādhyavasānaṃ syād āyur evedam īdṛśam //

upādānārpaṇadvāre dve cānye iti ṣaḍvidhā 
kuntā viśanti gaṅgāyāṃ ghoṣo nivasatīti ca //

lakṣyalakṣakavaiśiṣṭyavaiśiṣṭyād dvividhā punaḥ 
sarasaṃ kāvyam amṛtaṃ vidyā sthirataraṃ dhanam //

tathā sahetur atathā-bhedabhinnā ca kutracit 
saundaryeṇaiṣa kandarpaḥ sā ca mūrtimatī ratiḥ //

śabde padārthe vākyārthe saṃkhyāyāṃ kārake tathā 
liṅge ceyam alaṅkārā-ṅkurabījatayā sthitā //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke mahati navasaṃkhyaḥ sukhayatu //

dharmaṃ kaṃcit puraskṛtya prāyaḥ śabdaḥ pravartate 
yayārthaṃ spaṣṭam ācaṣṭe śabdas tām abhidhāṃ viduḥ //

jātyā guṇena kriyayā vastuyogena saṃjñayā 
nirdeśena tathā prāhuḥ ṣaḍvidhām abhidhāṃ budhāḥ //

gaur nīlaḥ pācako daṇḍī ḍitthaḥ kaṃsa iti kramāt 
kaṃ saṃhinasti kaṃsārir naraṃ ca kaṃ samāśritam //

na yogāder āyatanaṃ na saṅketaniketanam 
vṛttyā nirdeśaśabdo 'yaṃ mukhyayā svābhidheyayā //

pīyūṣavarṣaprabhavaṃ candrālokamanoharam 
sudhānidhānam āsādya śrayadhvaṃ vibudhā mudam //

jayanti yājñikaśrīmanmahādevāṅgajanmanaḥ 
sūktipīyūṣavarṣasya jayadevakaver giraḥ //

mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau 
mayūkhas tenāsau sukavijayadevena racite ciraṃ candrāloke mahati daśasaṃkhyaḥ sukhayatu //
