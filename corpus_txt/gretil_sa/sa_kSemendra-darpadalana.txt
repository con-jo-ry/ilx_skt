darpadalanam

prathamo vicāraḥ

praśāntāśeṣavighnāya darpasarpāpasarpaṇāt 
satyāmṛtanidhānāya svaprakāśavikāsine //

saṃsāravyatirekāya hṛtotsekāya cetasaḥ 
praśamāmṛtasekāya vivekāya namo namaḥ // (yugmam)

kṣemendraḥ suhṛdāṃ prītyā darpadoṣacikitsakaḥ 
svāsthyāya kurute yatnaṃ madhuraiḥ sūktibheṣajaiḥ //

kulaṃ vittaṃ śrutaṃ rūpaṃ śauryaṃ dānaṃ tapastathā 
prādhānyena manuṣyāṇāṃ saptaite madahetavaḥ //

ahaṃkārābhibhūtānāṃ bhūtānāmiva dehinām 
hitāya darpadalanaṃ kriyate mohaśāntaye //

kulaṃ kulaṃ kalayatāṃ mohānmithyābhimāninām 
lagnaḥ ko 'yaṃ na jānīmaḥ stabdhagrīvāgrahagrahaḥ //

kulasya kamalasyeva mūlamanviṣyate yadi 
doṣapaṅkaprastāntas tadāvaśyaṃ prakāśate //

yathā jātyaturaṃgasya na śakyajjātya(?)mucyate 
tathā guṇavataḥ sūnur nirguṇastatkulodbhavaḥ //

ekaścetpūrvapuruṣaḥ kule yajvā bahuśrutaḥ 
aparaḥ pāpakṛnmūrkhaḥ kulaṃ kasyānurvartatām //

loke kulaṃ kulaṃ tāvad yāvatpūrvasamanvayaḥ 
guṇaprabhāve vicchinne samāptaṃ sakalaṃ kulam //

kulābhimānaḥ kasteṣāṃ jaghanyasthānajanmanām 
kulakūlaṃkaṣā yeṣāṃ jananyo nimnagāḥ striyaḥ //

kulīnasya kulīnasya navadāridryalajjayā 
kiṃ kulenākulīnāgre yāñcādainyapralāpinaḥ //

guṇavatkulajāto 'pi nirguṇaḥ kena pūjyate 
dogdhrīkulodbhavā dhenur vandhyā kasyopayujyate //

svayaṃ kulakṛtastasmād vicārya tyajyatāṃ madaḥ 
guṇādhīnaṃ kulaṃ jñātvā guṇeṣvādhīyatāṃ matiḥ //

mūlānveṣaṇacintyamānamaniśaṃ nāstyeva puṃsāṃ kulaṃ strīṇāṃ yatra paramparaiva tanute saṃtānatantukramam 
etāsāṃ kṛtakaprapañcaracanālajjāvatīnāṃ puraḥ saṃsaktasmararūḍhagūḍhacaritaṃ tattvena jānāti kaḥ //

kulābhimānābharaṇasya mātā pitāmahī vā prapitāmahī vā 
yoṣitsvabhāvena yadi praduṣṭā tadeṣa doṣaḥ kulamūlaghātaḥ //

sūryavaṃśe triśaṅkuryaś caṇḍālo 'bhūnmahīpatiḥ 
dilīparaghurāmādyāḥ kṣitipāstatkulodbhavāḥ //

bhūbhujāṃ somavaṃśyānāṃ yaḥ pūrvapuruṣo budhaḥ 
gurutalpe sa candrasya jāto jagati viśrutaḥ //

kanyāyāstanayaḥ karṇaḥ kṣetrajāḥ pāṇḍunandanāḥ 
sāmānyakulacarcābhiḥ kimanyābhiḥ prayojanam //

mathurāyāmabhūtpūrvaṃ brāhmaṇaḥ śrīmatāṃ varaḥ 
yajvā śrutanidhirnāma śrutimānviśrutaśrutaḥ //

tasya muktālatā nāma prāṃśuvaṃśasamudbhavā 
babhūva vallabhā patnī lāvaṇyalalitākṛtiḥ //

tasyāṃ tasyābhavatkāntaḥ suvṛttaḥ saguṇāgraṇīḥ 
putrastejonidhirnāma vidyāvimaladarpaṇaḥ //

sa dhīmānvedavidvādī kaviḥ sarvakalālayaḥ 
sabhāsu viduṣāṃ cakre lajjayāvanataṃ śiraḥ //

taṃ darpadoṣajvaritaṃ grīvāstambhayutaṃ rahaḥ 
praśamāya pitā snehāt pathyaṃ vaktuṃ pracakrame //

putra mithyābhimānena kiṃ prayāto 'si mūḍhatām 
yanmadadviradārūḍhaḥ pūjyapūjāsu lajjase //

nāstyupāyaḥ sa saṃsāre darpaśvabhranipātinām 
mūḍhānāṃ kriyate yena kṣanaṃ hastāvalambanam //

kaṣṭaṃ kenopadiṣṭaste vinaṣṭavinayasmṛteḥ 
madaḥ sādhujanāviṣṭaḥ kulavidyādhanodbhavaḥ //

asthiraḥ kulasaṃbandhaḥ sadā vidyāvivādinī 
mado mohāya mithyaiva muhūrtanidhanaṃ dhanam //

etadeva kulīnatvam etadeva guṇārjanam 
yatsadaiva satāṃ satsu vinayāvanataṃ śiraḥ //

dayaiva viditā vidyā satyamevākṣayaṃ dhanam 
akalaṅkavivekānāṃ śīlamevāmalaṃ kulam //

abhogasubhagā bhūtir adainyadhavalaṃ kulam 
adarpaviśadā vidyā bhavatyunnatacetasām //

dveṣaḥ kasya na doṣāya prītiḥ kasya na bhūtaye 
darpaḥ kasya na pātāya nonnatyai kasya namratā //

tyāginā kiṃ daridreṇa kiṃ kulīnena pāpinā 
tuṣṭena kiṃ kadaryeṇa darpāndhena budhena kim //

vairāyate suhṛdbhāvaḥ pradānaṃ haraṇāyate 
darpabhūtābhibhūtasya vidyā maurkhyaśatāyate //

guṇināṃ matsaraḥ śatrur lubdhānāmatiyācakaḥ 
sarva eva sadarpāṇāṃ na kaścitpriyavādinām //

tasmātkāryastvayā putra nāhaṃkāraḥ kadācana 
darpodgrīvaḥ kilogreṇa mohagrāheṇa gṛhyate //

vaṃśenonnatiśālinā guṇagaṇenāntaścamatkāriṇā rūpeṇātimanohareṇa mahatā vittena vṛttena vā 
rohanmohamahātarurmadamayaḥ saṃjāyate yaḥ sadā tasyādau dṛḍharūḍhamūladalane kāryo 'bhiyogastvayā //

guṇeṣvanādaraṃ putra prāptaśrīrapi mā kṛthāḥ 
saṃpūrṇo 'pi ghaṭaḥ kūpād guṇacchinnaḥ patatyadhaḥ //

kulābhimānaṃ tyaja saṃvṛttāgraṃ dhanābhimānaṃ tyaja dṛṣṭanaṣṭam 
vidyābhimānaṃ tyaja paṇyarūpaṃ rūpābhimānaṃ tyaja kālalehyam //

putra prayatnena vibodhito 'si na muñcasi tvaṃ yadi darpamoham 
tadeṣa te yāsyati śalyabhāvaṃ tīvrābhitāpaprasavo 'bhimānaḥ //

vibhūtinalinīgajaḥ sujanamānabhaṅgaśaninipātapathadaiśikaḥ sukṛtacitradhūmodgamaḥ 
parāśayanavajvaraścaritacandrabimbāmbudaḥ sadā samadacetasāṃ guṇavināśaheturmadaḥ //

anityateyaṃ yadi nityatā syāt sarvaṃ na pāke virasaṃ yadi syāt 
kulārthavidyādikṛto 'bhimānas tadaiṣa te syānna viḍambanīyaḥ //

ahaṃ vādī vidyāparicayaguruḥ sarvaviduṣām ahaṃ mānī vāṇīprasaraparipākena sukaviḥ 
ahaṃ līlāhaṃsaḥ kuvalayadṛśāṃ mānasacaraḥ karotyantaḥ puṃsāmiti madapiśācaḥ paricayam //

lakṣmīḥ kṣaṇakṣayavatī parirakṣitāpi kāyo 'pyapāyanicayasya nikāya eva 
saṃbhogayogasukhasaṃgatirapyatathyā mithyābhimānakalanāghana eṣa śāpaḥ //

ityukto 'pyasakṛtpitrā līlāmīlitalocanaḥ 
sa yayau mattahastīva vegādagaṇitāṅkuśaḥ //

pādena kṣitimālikhanti samadāḥ kopoṣṇaniḥśvāsinas tiryagjihmanirīkṣaṇairvidadhati bhrūbhaṅgabhīmaṃ mukham 
sasvedāṅgulikandalīnikaṣaṇaistāmyallalāṭatvacaḥ kampante hitamantravādasamaye bhūtābhibhūtā iva //

sa kadācidvarāśveṣu sthiteṣu javakautukāt 
pratasthe kharamāruhya vayasyagṛhamutsave //

tena tīkṣṇapratodena codyamānaḥ punaḥ punaḥ 
kharastīvravyathārto 'bhūt prakṣaratkṣatajokṣitaḥ //

srutākṣaḥ kathitakleśaḥ svasvanocitasaṃjñayā 
so 'vadatsaṃmukhāyātāṃ gardabhīṃ jananīṃ nijām //

mātarbrāhmaṇaputro 'yaṃ paśya māmadhamāśayaḥ 
vidārayanpratodena vahantaṃ hantumudyataḥ //

kiṃ karomi yamenāhaṃ labdho 'nena durātmanā 
avaṭe pātayāmyenaṃ tanuṃ śvabhre kṣipāmi vā //

ityārtarāviṇaṃ putraṃ sāśrunetrātha gardabhī 
tamuvāca sasaṃtāpaṃ snehasaṃkrantatadyathā //

vahainaṃ durmadaṃ putra sahasva viṣamāṃ vyathām 
asya nāstyeva hṛdaye dāruṇe karuṇākaṇaḥ //

raudraḥ śūdreṇa jāto 'yaṃ brāhmaṇyāṃ brahmavarjitaḥ 
paraduḥkhaṃ na jānāti caṇḍaṃ caṇḍālaceṣṭitaḥ //

dayādaridraṃ hṛdayaṃ vacaḥ krakacakarkaśam 
yonisaṃkarajātānām etatpratyakṣalakṣaṇam //

navanītopamā vāṇī karuṇākomalaṃ manaḥ 
ekabījaprajātānāṃ bhavatyavanataṃ śiraḥ //

raṭati kaṭukāṭopaṃ kopādakāraṇavairavān spṛśati na dayāṃ dainyāpanne vijātitayā śaṭhaḥ 
kṣaṇarasikatālolaḥ sevāśritānavamanyate guṇiṣu kurute garvodgārānakharvagalaḥ khalaḥ //

iti duḥsahamākarṇya gardabhīvacanaṃ dvijaḥ 
sarvaprāṇisvanābhijñaḥ saṃmohābhihato 'patat //

sa labdhasaṃjñaḥ sucirān meruśṛṅgādiva cyutaḥ 
tatyāja sahasā darpaṃ naṣṭākhilakulonnatiḥ //

saṃmūrcchito viṣeṇeva sa gatvā māturantikam 
yathāśrutaṃ nivedyāsyai sarvaṃ papraccha tāṃ rahaḥ //

tanutyāgapravṛttena pṛṣṭā sā tena śāpitā 
adhomukhī tamavadad vailakṣyalulitākṣaraiḥ //

lajjākaramasatkarma kathaṃ tatkathayāmi te 
saṃsārādapi sāścaryaṃ gahanaṃ strīviceṣṭitam //

api kuñjarakarṇāgrād api pippalapallavāt 
api vidyudvilasitād vilolaṃ lalanāmanaḥ //

na bādhyante guṇaiḥ patyur na lakṣyante parīkṣakaiḥ 
na dhanena nivāryante śīlatyāgodyatāḥ striyaḥ //

dhanayauvanasaṃjātadarpakāluṣyaviplavāḥ 
kenonnataparibhraṣṭā vāryante nimnagāḥ striyaḥ //

dehapradāḥ prāṇaharā narāṇāṃ bhīrusvabhāvāḥ praviśanti vahnim 
krūrāḥ paraṃ pallavapeśalāṅgyo mugdhā vidagdhānapi vañcayanti //

ahaṃ pura rajaḥsnātā kāle kusumulāñchane 
ekākinī puṣpavane yauvanonmādinī sthitā //

vratadīkṣāpare patyau serṣyeva vinatānanā 
unnatastanavinyastahastā ciramacintayam //

etāḥ śvasanasotkampāḥ sajṛmbhāḥ ṣaṭpadasvanaiḥ 
sotkaṇṭhamiva gāyanti latāḥ puṣparajakhalāḥ //

udbhinnayauvanākrāntā priyabhogaviyoginī 
vrataśeṣajuṣaḥ patyur doṣeṇaivāsmi niṣphalā //

iti cintākṣaṇe tasmiṃl lagnābhimukhadarpaṇaḥ 
nāpitaḥ parihāsākhyaḥ śīlaśatrurivāyayau //

sa māmekākinīṃ dṛṣṭvā naṣṭasaṃvṛttikātarām 
pasparśotkampinīṃ pādanakhagrahaṇalīlayā //

tatrāhaṃ vṛttakartavyā nīcasaṃgamalajjayā 
adhomukhī cyutaṃ śīlaṃ vīkṣamāṇeva mūrcchitā //

avidūre carantī sā kharī sarvaṃ dadarśa tat 
gūḍhagarbhapradaṃ caitat karma me kulapātakam //

āstāṃ kimanayā putra guptavṛttāntacarcayā 
saṃvṛttānyeva śobhante śarīrāṇi kulāni ca //

iti māturvacaḥ śrutvā yātaḥ sa sahasāndhatām 
jātimānāvapatanān nirjīvita ivābhavat //

atha gatvā nirāhāraḥ sa kailāsāṭṭāhāsinīm 
āśāṃ brāhmaṇyabaddhāśaś cacāra suciraṃ tapaḥ //

tasyogratapasā tuṣṭaḥ svayameva śatakratuḥ 
brāhmaṇyaṃ yācamānasya na dadau durlabhaṃ bhuvi //

punaḥ punaḥ sa tapasā saṃtāpitajagattrayaḥ 
sahasrākṣavarātprāpa devatvaṃ na tu vipratām //

chandodevābhidhāno 'tha so 'bhavadbhuvi viśrutaḥ 
pratyabdamekadivase hy arcanīyo mṛgīdṛśām //

saṃmohapātālaviśālasarpas tasmānna kāryaḥ kulajātidarpaḥ 
śamakṣamādānadayāśrayāṇāṃ śīlaṃ viśālaṃ kulamāmananti //

mātā na yasyāstyavivekarāśiḥ punarbhavābdhirjanako na yasya 
yasya prasaktā dayitā na tṛṣṇā sa eva loke kuśalī kulīnaḥ //

dvitīyo vicāraḥ

dhanena darpaḥ kimayaṃ narāṇāṃ lakṣmīkaṭākṣāñcalacañcalena 
yatkaṃdharābaddhamapi prayāti naikaṃ padaṃ kālagatasya paścāt //

surakṣitaṃ tiṣṭhati nirnimittam arakṣitaṃ tiṣṭhati daivayogāt 
sthitaṃ kadaryasya ca copayuktam unmattanṛttopamameva vittam //

karmoktinarmanirmāṇaiḥ prātaḥ prātaḥ pradhāvatām 
dhanaṃ dhanaṃ pralapatāṃ nidhanaṃ vismṛtaṃ nṛṇām //

vicchāyayornirvyayayoḥ kaṣṭakliṣṭakalatrayoḥ 
viśeṣaḥ kleśadoṣasya kaḥ kadaryadaridrayoḥ //

ye dhanādānasaṃnaddhā nekṣante nidhanāvadhim 
nindanto lubdhatāṃ teṣām ante 'nye bhuñjate dhanam //

uktaṃ parasyāmiṣatām anuktaṃ yātyadṛśyatām 
hṛdaye śalyatāṃ dhatte nidhane dhanināṃ dhanam //

dhanena jīviteneva kaṇṭhasthena nirīkṣate 
paryante 'pyaprakāśena bandhūnāṃ mukhamāturaḥ //

yadarjitaṃ parikleśair arjitaṃ yanna bhujyate 
vibhajyate yadante 'nyaḥ kasyacinmāstu taddhanam //

vidyā vivādāya dhanaṃ madāya prajñāprakarṣaḥ paravañcanāya 
atyunnatirlokaparābhavāya yeṣāṃ prakāśastimiraṃ hi teṣām //

aśāntāntastṛṣṇā dhanalavaṇavārivyatikarair gatacchāyaḥ kāyaściravirasarūkṣāśanatayā 
anidrā mando 'gnirnṛpasalilacaurānalabhayāt kadaryāṇāṃ kaṣṭaṃ sphuṭamadhanakaṣṭādapi param //

śrāvastyāṃ sārthavāho 'bhūd arthanātha ivāparaḥ 
nando nāma nirānandaḥ kīrtanenārthināmapi //

sa kadaryaḥ sadā sarvajanasyodvegaduḥsahaḥ 
mūrdhaśāyī nidhānānāṃ kālavyāla ivābhavat //

kṛtvā samastaṃ divasaṃ dhanānāṃ nidhānakumbhīgaṇanāvidhānam 
sa lājapeyāpalamānaśīlam aśnāti rātrāvudaraṃ saśūlam (?) //

nirvyañjanaṃ nirlavaṇaṃ vinaṣṭam amṛṣṭapākaṃ viniviṣṭakaṣṭam 
adṛṣṭahāsaṃ vyayasaṃnirodhāt tasyābhavadveśma saśokamūkam //

vicchāyaṃ niḥsukhānandaṃ nirdīpaṃ jalavarjitam 
tasya kaṣṭaṃ kadaryasya paralokamabhūdgṛham //

sa bhaktasaṃcaye nityam abhaktaḥ saṃtatāmayaiḥ 
suvarṇavānvivarṇo 'bhūt saṃpūrṇaścintayā kṛśaḥ //

puṇyaprāpyā matirnāma dhanarddhiriva rūpiṇī 
bhāryābhūttadayogyasya tasya daivaviparyayāt //

sadā pracchādya sā bhartuś cakārātithisatkriyām 
tena vyayavivādeṣu śoṣitā kalahāgninā //

tasyāṃ tasyābhavatsūnuḥ saguṇaścandanābhidhaḥ 
pitrā lobhāndhakāreṇa nītaḥ padma ivānyatām //

kadācitsvagṛhadvāri dṛṣṭvā labdhānnamarthinam 
cakāra kalahaṃ nandaḥ patnyā śoṇitapātanam //

so 'vadatkopadaṣṭauṣṭhaḥ śvasanbhāryāmadhomukhīm 
tatsparsapāpaṃ stanayoḥ kṣālayantīmivāśrubhiḥ //

mama dāsyati ko bhikṣāṃ tvatpāṇikṣīṇasaṃpadaḥ 
dāridryajananī yasya sthitā tvaṃ durbhagā gṛhe //

striyo yatra pragalbhante bharturācchādya kartṛtām 
gṛhaṃ bhavatyavaśyaṃ tadāspadaṃ paramāpadām //

gṛhamekaṃ gṛhasthasya gṛhāṇāṃ śatamarthinaḥ 
bhāryābharjitavittasya naṣṭā gṛhapatergatiḥ //

tṛptidaṃ darśanenāpi jantorjīvitajīvitam 
draviṇaṃ yena rakṣanti svakāyaṃ bhakṣayanti te //

jīvannapyakriyo niḥsvaḥ śavo 'pyarthena sakriyaḥ 
dāridryaṃ maraṇaṃ loke dhanamāyuḥ śarīriṇām //

etadevārthasāmarthyaṃ pratyakṣeṇopalakṣyate 
yatskandhabandhe jīvadbhiḥ śavaḥ śibikayohyate //

prayacchasi kimarthibhyas tvamannaṃ kleśasaṃcitam 
dīyate yatkila prāptyai tatprāptaṃ kiṃ na rakṣyate //

putradārādisaṃbandhaḥ puṃsāṃ dhananibandhanaḥ 
kṣīṇātputrāḥ palāyante dārā gacchanti cānyataḥ //

paṇḍitāḥ kavayaḥ śūrāḥ kalāvantastapasvinaḥ 
vaidyasyeva savittasya vīkṣante mukhamāturāḥ //

iti tasya vacaḥ śrutvā kṛpaṇasyārthaniṣkṛpam 
sā tamūce samucitaṃ sattvasyābhijanasya ca //

santaḥ kurvanti yatnena dharmasyārthe dhanārjanam 
dharmācāravinīnānāṃ draviṇaṃ malasaṃcayaḥ //

yatkarotyaruciṃ kleśaṃ tṛṣṇaṃ mohaṃ prajāgaram 
na taddhanaṃ kadaryāṇāṃ hṛdayavyādhireva tat //

vardhate yo dhanavyādhiḥ sukhabhogaviyogakṛt 
tasyāśu śamanaṃ pathaṃ rājavaidyacikitsayā //

lobhānnābhūdgṛhe yasya kadācitkaścidutsavaḥ 
nṛtyanti paṭahaistasya nidhane dhanabhāginaḥ //

kaṇācāmatuṣāṅgārān yatnena parirakṣasi 
mūṣakāpahṛtaṃ koṣe ratnarāśiṃ na paśyasi //

dhanena darpaḥ ko nāma yatkṣaṇena vinaśyati 
rakṣyamāṇaṃ vyayenaiva bhakṣyamāṇamupaplavaiḥ //

vicāryamāṇastattvena daivādhīnatayā nṛṇām 
na kasyāṃcidavasthāyāṃ dhanalobhaḥ praśasyate //

kalau kāle khale mitre putre durvyasanānvite 
taskareṣu pravṛddheṣu lubdhe rājñi dhanena kim //

ṛṇikaiḥ kalahairnityam acchinnaguṇanāgateḥ 
dānadviṣo 'napatyasya mandāgneśca dhanena kim //

sahasāsāditārthasya rājadrohādipātakaiḥ 
bhayādavyayaśīlasya śalyeneva dhanena kim //

ghorapratigrahagrāmagrastodagraguṇaujasaḥ 
tadvibhāgānabhijñasya dhūrtāptasya dhanena kim //

rātrisevāvasannasya śītavātātapasthiteḥ 
prabhudṛṣṭiprahṛṣṭasya kaṣṭārhasya dhanena kim //

prabhūtalābhalobhena prayuktārthasya sarvataḥ 
bhūrjadṛṣṭena tuṣṭasya naṣṭabuddherdhanena kim //

malaśīlasya vaṇijasthūtkṛtasya jugupsayā 
laśunasyāśuceḥ pākagandheneva dhanena kim //

kāṅkṣitenāpyalabdhena bhogārhe navayauvane 
jarājīrṇaśarīrasya bhāreṇeva dhanena kim //

pravrajyātyaktagehasya janagauravapūjayā 
dhanasaṃghaṭitārthasya bandheneva dhanena kim //

śiśoraṅkuśaśūnyasya pātitasyāpathe viṭaiḥ 
kṣaṇakṣayopayogena svapneneva dhanena kim //

bhāryayā svairacāriṇyā grāmasthasya niyoginaḥ

adattabhuktamutsṛjya dhanaṃ sucirarakṣitam

tasya yātasya nirayaṃ nināya nṛpatirdhanam

babhūva bhūrisaṃbhārabhogavyayamahotsavaḥ //

bhogabhaṅgabhayeneti prātastatrābavījjanaḥ //

dhigdhigdhanaṃ kunidhanaṃ nandasyevātmabādhanam

andhaḥ kubjaḥ kṛśaḥ khañjaḥ kuṣṭhī sthūlagalagrahaḥ 
samūha iva duḥkhānāṃ sa tasyāstanayo 'bhavat //

tadapuṇyaiḥ parikṣīṇe mātuḥ kṣīre sa niścalaḥ 
kṛpayā bāndhavastrībhiḥ śunīkṣīreṇa vardhitaḥ //

etadeva viruddhānāṃ vaicitryaṃ pūrvakarmāṇām

vraṇaiḥ sa pūtikalilaklāmyatkṛmikulairvṛtaḥ 
pakvaṇe kuṇapākāras tasthau klinnatṛṇāstare //

tasminnapyativātsalyāt putrāsthāṃ jananī sthirām 
babandha vāsanālīnaḥ snehamoho hi duḥsahaḥ //

sa vardhamānaḥ śanakaiḥ smaśānāṅgāradhūsaraḥ 
pakvaṇograpiśācānām apyudvegakaro 'bhavat //

yaṣṭīniṣaṇṇagamanaḥ kuṣṭakledajugupsitaḥ 
sa jagāma pathā yena prayayau tena nāparaḥ //

atrāntare candanasya pituḥ śrāddhadine mahān 
babhūvārthisamūhānnadāne kalakalasvanaḥ //

tataḥ karparamādāya sa caṇḍālaśiśuḥ śanaiḥ 
ācāmayācakaḥ kṛcchrād dvārāgrabhuvamāyayau //

taṃ dṛṣṭvā candanaḥ saudhād viprāṇāṃ mārgadūṣaṇam 
nivāryatāmayaṃ prāptas tūrṇamityavadatkrudhā //

prabhubhrūbhaṅgabhītena laguḍenāhatastataḥ 
dvārapālena sāvartaḥ sa kapota ivābhavat //

sa nirbhinnalalāṭāsthiprakṣaratkṣatajokṣitaḥ 
kṣaṇaṃ saṃmūrcchitaḥ prāpa kleśabhogāya jīvitam //

adūravartinī śrutvā caṇḍālī tadyathāravam 
upasṛtya śuśocārtā spṛśantī tasya śoṇitam //

kena niṣkaruṇenedaṃ darśitaṃ bata pauruṣam 
praklinnakāyavikale yenāsminsubhaṭāyitam //

kāyāpāpamayīṃ duḥkhadaśāṃ dṛṣṭvāsya duḥsahām 
vairāgyāvasare kena krauryamevaṃvidhaṃ kṛtam //

ārtimevaṃvidhāmasya hṛdayakledinīmimām 
vilokya kuryātkaḥ pāpaṃ pāpaṃ hi padamāpadām //

yadyanena mahatpāpaṃ na kṛtaṃ pūrvajanmani 
taducyatāṃ sphuratkaṣṭā dṛṣṭā kasyedṛśī daśā //

ye dṛśyante vipatkleśaviśeṣaviṣamavyathāḥ 
ta eva guravaḥ pāpavāpasya (?) karaṇe nṛṇām //

karuṇārheṣu śūrāṇām upakāriṣu vairiṇām 
vañcakānāmapāpeṣu pāpasaṃkhyāṃ karoti kaḥ //

tāraṃ rodiṣi kiṃ putra sahasvāghātajāṃ rujam 
aśarmakarmanirmāṇaṃ marmacchedi śarīriṇām //

iti tasyāṃ pralāpinyāṃ prekṣavāpte jane jinaḥ 
anāthabandhuḥ karuṇāsindhustenāyayau pathā //

bhavabhramāsaktapariśramāṇāṃ rāgādidoṣairupatāpitānām 
āśvāsanenāmṛtasodareṇa limpanniva dyāṃ dyuticandanena //

dṛṣṭvā tamāpadgatamugrarogabhagnaṃ nimagnaṃ vyasane vivignam 
vyalambatārdraḥ karuṇārasena tattāpaśāntyai bhagavāñjinendraḥ //

tatsaṃnidhānena muhūrtamātraṃ sa nirvyathaḥ svāsthyamivāsasāda 
nihanti pāpaṃ kuśalaṃ prasūte saṃdarśanaṃ sattvahitāśayānām //

jñātvātha candanaḥ prāptaṃ bhagavantaṃ tathāgatam

haimaṃ kamalamāruhya tasthau paryaṅkalīlayā //

praṇataṃ caraṇālīnaṃ pūjāvyagrakaraṃ puraḥ 
babhāṣe bhagavānprīto bhikṣusaṃsadi candanam //

kimayaṃ yācamāno 'pi varākastāḍitaḥ krudhā 
kṛtaṃ na kṛpaṇe kasmāt karuṇākomalaṃ manaḥ //

dayārdrāḥ sarvasattveṣu bhavanti vimalāśayāḥ 
evaṃvidhānāṃ duḥkhānāṃ kāraṇaṃ kaluṣaṃ manaḥ //

kṛtakrūrāpakāreṣu vidveṣaparuṣeṣvapi 
bhavanti santaḥ kleśoṣmaśoṣiteṣu na karkaśāḥ //

kliṣṭaḥ kaṣṭaṃ kadaryo 'yaṃ lobhenāparajanmani 
apradānodyatenādya kāyakleśena pīḍitaḥ //

eṣa nandastava pitā pūrṇārthamalasaṃcayāt 
āvṛtaḥ pāparogeṇa caṇḍālatvamupāgataḥ //

janmāntare 'pyato 'nyasmin rogayogānmumūrṣaṇā 
suvarṇaṃ dattametena tenāyaṃ sadhano 'bhavat //

antyakleśadaśāyāṃ yan mumūrṣuḥ saṃprayacchati 
taccābhogyaṃ bhavatyasya lobhādanyeṣu janmasu (?) //

yaiḥ saṃcayosāharasaiḥ pranṛttaṃ śocanti te pātakamātmavṛttam //

ityuktvā bhagavānpuṇyāṃ vidadhe dharmadeśanām

prakṣālya na parīkṣyante khalairbhūpālasaṃsadi //

vādibhiḥ kalahodarkatarkasaṃparkakarkaśā

nānāviparyayapathair gacchantyarthaphalārthinaḥ //

patrasaṃcayasaṃbhāraiḥ kiṃ tayā bhāravidyayā //

āśāpāśāvalambinyā kiṃ tayā lubdhavidyayā //

yantratantrānusāriṇyā kiṃ tayā vyājavidyayā //

hāsyahetuḥ kathaṃ setuḥ sikatāmuṣṭibhirbhavet //

aho paropadeśeṣu sarvo bhavati paṇḍitaḥ //

yathā tvaṃ niṣphalārambhas tathāhamaparo jaḍaḥ //

tathāpi dṛḍhasaṃkalpaḥ svakṛtyānna cacāla saḥ //

sarvavidyānidhiryena sahasaiva babhūva saḥ //

nijāṃ tapaḥphalāvāptikathāṃ pitre nyavedayat //

bharadvājaḥ pramode 'pi svedākula ivāvadat //

kiṃ tvāgāmibhayādetan na yuktaṃ pratibhāti me //

itaḥ samīpe raibhyasya kopanasya tapovanam 
vidyāmadāndhau tatputrāvarvāvasuparāvasū //

tāvaśrāntaśrutonmādau tvaṃ cābhinavapaṇḍitaḥ 
tatsaṃgame dveṣamayaḥ sadā saṃnihitaḥ kaliḥ //

grīvāstambhabhṛtaḥ paronnatikathāmātre śiraḥśūlinaḥ sodvegabhramaṇapralāpavipulakṣobhābhibhūtasthiteḥ 
antardveṣaviṣapraveśaviṣamakrodhoṣṇaniḥśvāsinaḥ kaṣṭā nūtanapaṇḍitasya vikṛtirbhīmajvarārambhabhūḥ //

tava tatra prayātasya yuktāyuktavivādinaḥ 
bhaviṣyati muneḥ śāpād avaśyaṃ madanigrahaḥ //

śuktikārajatajñānanīlapītādidarśanaiḥ 
unmādaṃ janayatyeva vidyādarpapiśācikā //

eṣa vidyopadeśena vināśaḥ prārthitastvayā 
raibhyāśramo na gantavyaḥ kartavyaṃ yadi madvacaḥ //

ityukto 'pyasakṛtpitrā sa gatvā raibhyaputrayoḥ 
vyadhādvivādanirvedaiḥ sadā vidyāmadakṣitim //

taṃ darpamattaṃ sākopabhīmabhrūbhaṅgadurmukhau 
tāvūcaturmanaḥsaktavidyāvidveṣaśūlinau //

kanīyānāvayoryasmād vayasā tvaṃ śrutena ca 
karoṣi vādairākṣepaṃ tasmādāyuḥkṣayo 'stu te //

ityukto 'pi krudhā tābhyāṃ na darpādvirarāma saḥ 
na prasannaṃ na ca kruddhaṃ gaṇayanti madoddhatāḥ //

atrāntare bhramadbhṛṅgamālābhrūbhaṅgavibhramaḥ 
kālaḥ proṣitakāntānāṃ puṣpakālaḥ samāyayau //

kṣiptapattrāḥ sumanasāṃ rajaḥkaluṣitekṣaṇāḥ 
sadveṣā iva vidvāṃsaś cerurmalayamārutāḥ //

mādhuryalalitodāravāṇīvilasitairmuhuḥ 
kavīnāmiva saṃgharṣaḥ kokilānāmajāyata //

raibhye prayāte putrābhyāṃ saha snātuṃ sarittaṭam 
bharadvājātmajo 'bhyetya praviveśa tadāśramam //

tatra puṣpoccayavyagrāṃ dharmapatnīṃ purāvasoḥ 
so 'paśyatsuprabhāṃ nāma rūpadarpāpahāṃ rateḥ //

uṭajāṅganasaktānāṃ hariṇīnāṃ vilokane 
vilāsadīkṣāṃ kurvāṇāṃ taralāpāṅgamaṅgibhiḥ //

tāṃ dṛṣṭvā candravadanāṃ madanānandadevatām 
babhūvotkrāntamaryādaḥ sahasaiva muneḥ sutaḥ //

sa brahmacārī kāmena navena taralīkṛtaḥ 
abhilāṣocitaṃ vaktum anabhijño 'pyuvāca tām //

unmādanamidaṃ rūpam anurūpaṃ manobhuvaḥ 
samutsiktamivāsaktaṃ karoti mama mānasam //

vidyāvinayamutsṛjya saṃtyajya guruyantraṇām 
tvayi pravṛttaṃ cittaṃ me prāgjanmapremabandhanam //

jānāmi yatkṛtasyāsya vipāke karmaṇaḥ phalam 
tathāpyabhimataṃ dhartuṃ na śaknomi karomi kim //

na śrutena na vittena na vṛttena na karmaṇā 
pravṛttaṃ śakyate roddhuṃ manobhavapathe manaḥ //

ityuktvā tāṃ bhayodbhrāntanayanāmāśramonmukhīm 
gantuṃ pravṛttāṃ so 'bhyetya jagrāhāṃśukapallave //

kadalī kuñjareṇeva tarasā tena nirjane 
kṛṣyamāṇā tamavadat sā niṣedhacalāṅguliḥ //

mā mā malinaya svacchaṃ śīlaṃ mama tathātmanaḥ 
vidyāyā niravadyāyāḥ kimetaducitaṃ phalam //

śīlaśuklāṃśukāṃ tyaktvā lajjāṃ nijavadhūmiva 
gṛhṇāsi paranārīṇāṃ pāṇinā paṭapallavam //

kimetadityanucitaṃ dṛṣṭvā nūnaṃ kamaṇḍaluḥ 
udgrīvaḥ kautukeneva mukhaṃ tava nirīkṣate //

bibhrato 'ntargatarasāṃ kusumeṣuruciṃ navām 
jaṭāvalkalabhāraste taroriva na śāntaye //

pāpasaṃkalpamātreṇa trapayādhomukhī tava 
patitā sparśabhītyeva kampalolākṣamālikā //

āsanābje sarasvatyā japalolaradacchade 
durnayoktirna yukteyaṃ mukhe tava manīṣiṇaḥ //

iyaṃ tapovanamahī vivekajananī katham 
janayatyabhilāṣaṃ te jananīvājitātmanaḥ //

durmado (durdamo) yauvanabharas turaṅga iva hārakaḥ 
sarvathā śithilātmānam avaṭe kṣipati kṣaṇāt //

dhigdhiyaṃ kiṃ vivekena dūre viśrāmyatu śrutam 
dhāryate yairna saṃsāravikāraskhalitaṃ manaḥ //

kva vidyā viditāśeṣakāryākāryavimarśadhīḥ 
mūḍhatā kva ca duṣkarmamahāpāpakuṭumbinī //

ityucyamāno 'pi yadā na sa tatyāja durgraham 
śīlāpahārasaṃtrastā sā tadā samacintayat //

kiṃ karomyajane labdhā vivaśāhaṃ pramādinā 
utsṛṣṭadharmanimayāḥ kiṃ na kurvantyavāritāḥ //

ayaṃ smarāturastāvad vacasā na nivartate 
vañcyante sāntvavādena kāmakrodhamadoddhatā //

iti dhyātvā tamavadat sā śanairmṛduvādinī 
gaccha tvaṃ svayameṣyāmi niśi śūnyalatāgṛhe //

snātvā saputraḥ kāle 'smin nāyāti śvaśuro mama 
jvalajjvalanatulyasya tasyāgre kiṃ kariṣyasi //

ityuktaḥ sa tayā prāyāt satyaṃ vijñāya tadvacaḥ 
duṣprāpamapi manyante sulabhaṃ kāmamohitāḥ //

raibhyaṃ tataḥ samāyātam agnyāgārāgrataḥ sthitam 
snuṣā provāca kopāgnidhūmenevāśruvarṣiṇī //

bharadvājātmajastāta pāpastava suhṛtsutaḥ 
mamādya vijane śīlaviplave 'bhyarthitāṃ gataḥ //

sa mayā durgrahagrastaḥ sameṣyāmīti vañcitaḥ 
vimucye nānyathā hastāt tasya svastimatī satī //

etadākarṇya sahasā prajvalanmanyunā muniḥ 
babhūva durnimittolkāpātakrūre ivāṃśumān //

vidyāvatāṃ sphuratyantarvivekaḥ svasthacetasām 
vikārakāle saṃmohaś citte vidyā ca pustake //

sa niḥśvasannatha krodhajvarārambhāruṇekṣaṇaḥ 
abhicārajapeneva kampamānādharo 'bhyadhāt //

aho bata bharadvājaḥ putrasyādhyayane vyadhāt 
dharmopadeśaṃ yatnena nagnīkartuṃ parāṅganaḥ //

ityuktvāmarṣasaṃrambhād aparaṃ vaktumakṣamaḥ 
sa praviśyāgnisadanaṃ pratīkāraparo 'bhavat //

utpāṭya vikaṭāṭopakopaḥ prauḍhāgnipiṅgalam 
sa juhāva jaṭāṃ vahnau krūrakrodhasaṭāmiva //

dvitīyāyāṃ hutāyāṃ ca śūlabhṛdghorarākṣasaḥ 
kṛtyāsakhaḥ samudbhūtaḥ provāca praṇato munim //

kiṃ karomi mune kasya vināśāyāsmi nirmitaḥ 
trailokyamapi nirdagdhuṃ saṃnaddho 'haṃ tvadājñayā //

iti bruvāṇaṃ taṃ raibhyaḥ krūrākāramabhāṣata 
bharadvājasutaṃ gaccha kavalīkurvapaṇḍitam //

iti tena samādiṣṭaḥ sa vrajankampitāvaniḥ 
ardhaśaucaṃ munisutaṃ dṛṣṭvā durātsamādravat //

tasminnabhidrute vegād dīptaśūle niśācare 
bhayabhagnagatiḥ prāpa śaraṇaṃ na muneḥ sutaḥ //

palāyamānaḥ saṃprāptaḥ sa javātpiturāśramat 
agnyāgāraṃ viśanruddhaḥ śūdreṇāśaucadūṣitaḥ //

dāsaspṛṣṭaḥ sa niḥśaucaḥ patitaḥ saṃbhramātkṣitau 
rakṣaḥ śūlahataḥ paścāt sahasā bhasmasādabhūt //

atrāntare bharadvājaḥ praviśannijamāśramam 
vidhvastacchāyamālokya sodvegaḥ samacintayat //

mama puṣpaphalādānapratyāvṛttasya vahnayaḥ 
sadottiṣṭhanti puratas te 'dya kiṃ niścalā iva //

iti saṃcintya dṛṣṭvāgre bhasmabhītaṃ sutaṃ muniḥ 
śrutvā ca dāsakathitaṃ vṛttāntaṃ nyapatadbhuvi //

sa labdhasaṃjñaḥ śanakair avadadbāṣpagadgadam 
raibhyo 'pi vidvānkālena prāpnotu svasutādvadham //

hā putra rakṣitenāpi kṣaṇakṣayanipātinā 
na jīvāmi sadoṣeṇa kāyeneva tvayā vinā //

ityuktvā putraśokena citāgnimaviśanmuniḥ 
mahatsvapi navotsekād abhagnaprasarāḥ śucaḥ //

atha yāte śanaiḥ kāle bṛhadyumnasya bhūpateḥ 
yājakau jagmaturgeham arvāvasuparāvasū //

pravṛtte vidhivattasya dīrghasattre pṛthuśriyaḥ 
dānamānodayaḥ ko 'pi tayoryājakayorabhūt //

kadāciddinaparyantasaṃdhyāyāṃ nijamāśramam 
parāvasuḥ samāgacchan dṛṣṭvā pitaramagrataḥ //

kṛṣṇājinottarāsaṅgaṃ daṇḍena mṛgaśaṅkayā 
jaghāna śāpavivaśaḥ sa tenābhūdvicetanaḥ //

janakaṃ hatamālokya brahmahatyābhayākulaḥ 
gatvā yajñabhuvaṃ bhrātre sa tamarthaṃ nyavedayat //

arvāvasustamavadad bhrātaḥ kiṃ kriyate vidheḥ 
bhavanti yasya saṃkalpād evaṃrūpā viparyayāḥ //

dharmārthī pāpamāpnoti śīlārthī śīlaviplavam 
vidhau vidhuratāṃ yāte draviṇārthī daridratām //

brahmahatyāvrataṃ tīvraṃ bhavato 'rthe carāmyaham 
tvamasya kuru bhūbhartuḥ saṃpūrṇāṃ yājanakriyām //

uktvetyarvāvasurbhrātuḥ pāpaśāntyai dhṛtavrataḥ 
cakāra sarvatīrtheṣu tīvraniṣkṛtipāraṇam //

taṃ samāptavrataṃ prāptaṃ rājño yajñavasuṃdharām 
dūrātparāvasurjñātvā pitṛghnaḥ samacintayat //

ayaṃ me dakṣiṇākāle bhāgahartā samāgataḥ 
madabhāgyaiściraṃ tīvravratakliṣṭo 'pi jīvati //

iti saṃcintya so 'bhyetya provāca pṛthivīpatim 
lobhamātsaryayoraṅke patitaḥ pātakecchayā //

rājanyajñamahīmeṣa kilbiṣī brahmahatyayā 
praviśatyavikalpena madbhrātā vāryatāmitaḥ //

ityuktastena nṛpatiḥ kṛtaghnena viparyayāt 
tasya praveśamajñānān niṣpāpasya nyavārayat //

andhā iva na paśyanti yogyāyogyaṃ hitāhitam 
pathā tenaiva gacchanti nīyante yena pārthivāḥ //

mithyāpavādadānena naiva bhrātre cukopa saḥ 
nikāre kāraṇaṃ daivaṃ manyante hi manīṣiṇaḥ //

tena tasyānṛśaṃsyena nirvikāratayā tayā 
tuṣṭāḥ kratusamāsīnās tamūcustridivaukasaḥ //

praśamena tavānena prasannāste vayaṃ mune 
varārho 'pi varācāra gṛhyatāṃ pravaro varaḥ //

ityuktaḥ sa suraiḥ prītyā tānuvāca kṛtāñjaliḥ 
yadi yuṣmadvarārho 'haṃ dīyatāṃ yanmamepsitam //

matpitrā yo 'bhicāreṇa bharadvājātmajo hataḥ 
sa jīvatvasmṛtakrūranikāraḥ sa ca tatpitā //

asmatpitā mṛgadhiyā yaḥ parāvasunā hataḥ 
so 'pi vismṛtatatkopaḥ svasthaḥ prāpnotu jīvitam //

ityarthite vare tena tathetyākhyāyi taiḥ suraiḥ 
yavakrītabharadvājaraibhyāḥ prāpuḥ svajīvitam //

ityete munayo 'pi darpaviphale yāte śrute śocyatāṃ krodhāndhyena punaḥ pranaṣṭavimalāloke viveke cyute 
śīle rāgamahoṣmaṇā vigalite dveṣeṇa nāśaṃ gatāḥ kasyānyasya dhanābhimānamalinā vidyā vidhatte guṇam //

cetaḥ śāntyai dveṣadarpojjhitena yatnaḥ kāryaḥ sarvathā paṇḍitena 
vidyādīpaḥ kāmakopākulākṣṇāṃ darpāndhānāṃ niṣphalāloka eva //

alobhaḥ paramaṃ vittam ahiṃsā paramaṃ tapaḥ 
amāyā paramā vidyā niravadyā manīṣiṇām //

śukrasya vidyā dhanadārthahartum āryāprapañcopacitasya śocyā 
kacasya vācaspatijanmano 'pi vyājena vidyā viphalībabhūva //

spṛśati matiṃ nahi teṣāṃ dveṣaviṣaḥ kalisarpaḥ /
yadi śamavimalamatīnāṃ svamanasi bhavati na darpaḥ //

caturtho vicāraḥ

padmopamānāṃ dinasundarāṇāṃ ko 'yaṃ nṛṇāmasthirarūpadarpaḥ 
rūpeṇa kāntiḥ kṣaṇikaiva yeṣāṃ hāridrarāgeṇa yathāṃśukānām //

paryantarekhāṅgavibhāgahīnacitropamaṃ bālavapuḥ prakṛtyā 
tadyauvanenaiva vikāsameti caitrotsaveneva śirīṣapuṣpam //

alomaśaṃ pūrṇaśaśāṅkaśobhaṃ mukhaṃ tu yūnāṃ katiciddināni 
jāte tataḥ śmaśruviśālajāle śevālalīnābjatulāṃ bibharti //

dhūmena citraṃ tuhinena padmaṃ tamisrapakṣeṇa sudhāṃśubimbam 
śītaṃ nidāghena na bhāti toyaṃ jarāvatāreṇa ca cārurūpam //

rūpaṃ kṣaṇasvīkṛtaraktamāṃsagrāsaprasaktākṛtakāmadoṣā 
keśagraheṇaiva jarā janānāṃ veśyeva vittaṃ kavalīkaroti //

pākakrameṇaiva vicitrakarmā pratikṣaṇaṃ dehabhṛtāmalakṣyaḥ 
karoti kālaḥ pariṇāmaśaktyā rūpaṃ virūpaṃ caturapravāhaḥ //

na lakṣyate kālagatiḥ savegacakrabhramabhrāntividhāyinīyam 
hyo yaḥ śiśuḥ sa sphuṭayauvano 'dya prātarjarājīrṇatanuḥ sa eva //

puṃsāmavasthātritayatribhāge rūpapradaṃ yauvanameva nānyat 
tasminmadonmādagadāṅgabhaṅgavyaṅgyādidoṣopahate kva rūpam //

yadā naraḥ śocati duḥkhataptas tyaktāśanaḥ śokavivarṇavaktraḥ 
na snāti nottiṣṭhati naiva śete tadā kva rūpaṃ kva ca yauvanaśrīḥ //

yadā sthitaḥ preta ivāsthiśeṣaḥ kārāgṛhe dhūsaritordhvakeśaḥ 
prakīrṇayūkāmalakālakāyas tadā kva rūpasya gato 'bhimānaḥ //

yadā sadāṅgīkṛtadainyaduḥkhasevāpravāsena vinaṣṭakāyaḥ 
nityapravāsabhramabhagnajānur na rūpalabdhasya tadāsti rupam //

yadā prahārairdalitākhilāṅgaḥ khaṇḍoṣṭhanāsaḥ sphuṭitākṣidantaḥ 
yuvā piśācatvamivopayāti tadāpi rūpaṃ vigatasvarūpam //

yadā na dhīmānariṣu pramāthī na vākpaṭuścitramanuṣyatulyaḥ 
tadā surūpādavicāraramyād varaṃ virūpaḥ spṛhaṇīyarūpaḥ //

yadā daridraḥ paridhānahīnas trapānilīnaḥ kurute 'tiyāñcām 
kapolasaṃjātavalīvikāras tadā surūpo 'pi paraṃ virūpaḥ //

vidvatsaṃsadi vādibhiḥ kavivarairbhāṣānabhijñaḥ paraṃ mūrkhaḥ śaṃkaravāhanastutipadairyaḥ saṃjñayā hasyate 
vikrītaḥ paradeśapaṇyasadane dhūrtairivānuttaraḥ puṃsaścitramayūracāruvapuṣaḥ kiṃ tasya rūpaśriyā //

kālaṃ muhūrtāṅgulimaṇḍalena dinatriyāmāñjalinā pibantam 
rūpaṃ vilokyaiva vapuśca keṣāṃ bhaṅgena nāṅgānyalasībhavanti //

rūpaṃ vayaḥ śauryamanaṅgabhogaṃ prajñāprabhāvaṃ vibhavaṃ vapuśca 
aśnāti kālabhramaraḥ samantāt puṃsāṃ ni kiṃjalkamivāmbujānām //

kadācitsaha gandharvaiḥ sabhāsthāne śacīpatim 
nṛttenāpsarasaḥ sarvā gītena ca siṣevire //

tāsāṃ madhye babhau kāntā vṛttīnāmiva kaiśikī 
urvaśī svamukhe maitrīṃ vadantīvendupadmayoḥ //

śakrasevāgatāstatra tāṃ dṛṣṭvendumukhīṃ surāḥ 
menire dhanyamātmānaṃ śṛṅgārasyāṅgatāṃ gatam //

nṛtyantī sā babhau hāramadhyaratneṣu bimbitā 
yugapatpraviśantīva hṛdayāni divaukasām //

līnā devavimāneṣu haṃsāstadgatinirjitāḥ 
tatkaṭākṣajitaścakre nidrāṃ candre mṛgaḥ kṣaṇam //

tasyāḥ serṣyāpsaronetramāleva patitā babhau 
stanayoḥ śekharasrastā nīlotpaladalāvalī //

utsāhoddhatavibhramabhramarakavyāvṛttahārāntaratruṭyatsūtravimuktamauktikabharaḥ saktaḥ stanotsaṅgayoḥ 
vaktrenducyutasaṃtatāmṛtakaṇākāraścakāra kṣaṇaṃ tasyā nṛttarasaśramoditaghanasvedāmbubimbaśriyam //

tasyā nṛttavilokane pulakitaṃ dṛṣṭvā ratirmanmathaṃ niḥśvāsāñcitacārurūpa(?)rajasā cakre purastātpaṭam 
udvīkṣyākṣiparamparāmapi harestatrāvasannā śacī kopāndolitakelipadmamadhupairmadhye 'ndhakāraṃ vyadhāt //

vighnaṃ na cakrurnanu nṛttalīlāsaṃdarśane puṇyavatāṃ narāṇām /
tatrorvaśīrūpavaśīkṛtānāṃ nimeṣaśūnyāni vilocanāni //

devayoraśvinostatra rūpamādhuryadhuryayoḥ 
mithaḥ kathā samabhavat tadguṇākṛṣṭacittayoḥ //

eko 'bravīdaho rūpam asyāstaralacakṣuṣaḥ 
nimīlanniyamā yena munayo 'pyākulīkṛtāḥ //

asyāṃ saṃsadi kasyāsye patantyetāḥ sujanmanaḥ 
smarasaṃbhogasaṃvādalajjākuṭilitā dṛśaḥ //

vṛttasaṃgamayoreva parasparavilokane 
nyāsaṃ śṛṅgārasarvasvam anaṅgenārpitaṃ rahaḥ //

raṇotsṛṣṭatanoḥ kaṇṭhe sotkaṇṭhā bhujabandhanam 
kasyeyaṃ taralāpāṅgā raṅgottīrṇā kariṣyati //

iti bruvāṇamaparaḥ sasmitastamabhāṣata 
aho nu vismṛtaḥ kiṃ te bhūtalenduḥ purūravāḥ //

vikramābharaṇaṃ dikṣu lāvaṇyatilakaṃ bhuvaḥ 
urvaśībhogasubhagaṃ yasyaitadgīyate yaśaḥ //

tena rūpaguṇotsāhair urvaśīyaṃ vaśīkṛtā 
puraḥ sthitāpi śakrasya manasā tatra tiṣṭhati //

rūpasāmyena śītāṃśuvaṃśe jātaḥ sa lajjate 
na karoti rateragre tatkathāṃ matsarī smaraḥ //

na jāne bata hevākaḥ ko 'yaṃ kusumadhanvanaḥ 
naivārpayati yatpāṇau tasyaiva śarapañcakam //

bhuvaḥ samastāmbudhimekhalāyā voḍhāramājānuvilambibāhum 
līlāguruṃ taṃ hṛdaye vahantī tanvī kathaṃ nṛtyati naiva vidmaḥ //

draṣṭavyaḥ sa nṛpastāvad aprastāve 'pi yatnataḥ 
ko vetti tadvidhaṃ ratnaṃ puṇyairāste kiyacciram //

iyuktvā tau kṛtakṣoṇīpatidarśananiścayau 
nṛtte nivṛtte jambhāriṃ praṇamya yayaturbhuvam //

rājadhānīṃ samāsādya tau purūravasaḥ kṣaṇāt 
avāritau viviśatur vetribhiḥ suragauravāt //

tau taṃ dadṛśatuḥ snānavihitābhyaṅgasaṃgamam 
pīyūṣanavanītena lagnasnehamivoḍupam //

snātottāritakeyūramahārhamaṇikaṃkaṇam 
lāvaṇyābharaṇaṃ tasya virarājorjitaṃ vapuḥ //

śūnyaśravaṇapāśasya tasya kaṇṭhaḥ samāyayau 
nirbhūṣaṇaniveśo 'pi viśeṣaramaṇīytām //

vicārya tasyā maryādaṃ saundaryodāryamaśvinau 
praśaśaṃsaturāścaryanirmāṇātiśayaṃ vidheḥ //

sa tau kṛtāñjaliḥ prītyā kṛtāsanaparigrahau 
papraccha svacchahṛdayastv arāgamanakāraṇam //

tāvūcatuḥ kṣitipate mahīkusumadhanvanaḥ 
trailokyābharaṇaṃ rūpaṃ tavāvāṃ draṣṭumāgatau //

nisargeṇa jagatsarganirargalaguṇādarāt 
kautukālokasāreva dṛṣṭvā sṛṣṭiḥ prajāpateḥ //

vilokitastvaṃ vasudhāsudhāṃśuḥ pūrṇamaṇḍalaḥ 
rūpapīyūṣapānena prāptā prītiḥ kimucyate //

ityuktaḥ praṇayāttābhyāṃ kiṃcitkusumitasmitaḥ /
tāvūce nṛpatirmānyamānenābhyadhikāraḥ //

bhavatsaṃdarśanenāham asmyanugrahabhājanam 
draṣṭavyā draṣṭumāyānti puṇyapuṇyena kevalam //

tīrthāptiḥ sādhusaṃparkaḥ pūjyapūjāmahotsavaḥ 
asminvirasaniḥsāre saṃsāre sārasaṃgrahaḥ //

snānābhyaktena na mayā yuvayorucitaḥ kṛtaḥ 
puṇyasāphalyaniḥśalyakalyāṇāyārcanādaraḥ //

agnyāgārāntare tāvan muhūrtaṃ kriyatāṃ sthitiḥ 
kṛtasnānaḥ sameṣyāmi pūjāpraṇayapātratām //

iyuktau tena yayatus tau hutāśanamandiram 
snātaṃ vibhūṣitaṃ bhūpaṃ drakṣyāva iti kautukāt //

atha rājā kṛtasnānaḥ sarvābharaṇabhūṣitaḥ 
purohitena sahitas tatsamīpamupāyayau //

tau dṛṣṭvā pṛthivīpālaṃ tārahāraṃ kirīṭinam 
kṣaṇaṃ naivocatuḥ kiṃcid viṣaṇṇau vinatānanau //

kṛtārcane narapatau tau papraccha purohitaḥ 
akasmādyuvayoḥ kasmād aprasāda ivekṣyate //

vinayātikramo 'smākaṃ yātaḥ kacinna hetutām 
pṛṣṭau purohiteneti tau śanaistamabhāṣatām //

āvayornāprasannatvaṃ na yuṣmākamatikramaḥ 
kiṃ tu kālagalatsarvabhāvālokanavismayaḥ //

adhunaiva narendro 'yaṃ dṛṣṭo 'bhyaṅge 'pi yādṛśaḥ 
kṣaṇapākena kālasya dṛśyate naiva tādṛśaḥ //

dinendhanavane nityaṃ dahyamāne 'rkavahninā 
nīyate kāladhūmena rūpacitramacitratām //

niścitya sarvabhāvānāṃ nityametāmanityatām 
rūpe 'bhimānaṃ kaḥ kuryāt svapnacitrapaṭopame //

jarājīrṇāni rūpāṇi rogārtāni vapūṃṣi ca 
āyūṃṣi kālalīḍhāni dṛṣṭvā kasya bhavenmadaḥ //

yo 'yaṃ vikokyate lokaḥ sphārākāravikāravān 
ucchūnatāmupagatās ta ete śukrabindavaḥ //

aho kālasya sūkṣmo 'yaṃ ko 'pyalakṣyakramaḥ kramaḥ 
yatpākapariṇāmena sarvaṃ yātyanyarūpatām //

rājñaḥ snānakṣaṇe yābhūl lāvaṇyalaharī tanoḥ 
pītā kṣaṇena sā tena pravṛttānyakṣaṇocitā //

saṃpūrṇasyāyuṣo mātrā rūpasya vibhavasya ca 
horāyātrāmbudhāreva galatyevāniśaṃ nṛṇām //

kalākāṣṭhāmuhurtānāṃ kālasya vrajatāṃ javāt 
na lakṣyate vibhāgena dīpasyevārciṣāṃ gatiḥ //

bālaḥ prabhāte madhyāhne taruṇaḥ sthaviro 'stagaḥ 
dine dine dineśo 'pi kriyate kālalīlayā //

śuṣyantyambudhayastaraṅgagahanairāliṅgitāśāṅganā gacchantyudgatatuṅgaśṛṅgamukuṭodagrā girīndrāḥ kṣayam 
bhraśyatyeva vasuṃdharāpi sahitā digdantibhiryadvaśāt sarvāśī satataṃ pradhāvati mahākālaḥ sa ko 'pyākulaḥ //

ityuktvā nṛpamāmantrya divaṃ jagmaturaśvinau 
nṛpaśca tadvacaścintāśāntarūpamado 'bhavat //

tasmānna kāryaḥ sudhiyā vicārya sāścaryasaundaryavilāsadarpaḥ 
saṃsāramohaprasare ghane 'smin vidyullatāvisphuritaṃ hi rūpam //

prātarbālataro 'tha kudmalatayā kāntākucābhaḥ śanair helāhāsavikāsasundararuciḥ saṃpūrṇakoṣastataḥ 
paścānmlānavapurvilolaśithilaḥ padmaḥ prakīrṇe 'nilais tasminneva dine sa paṅkakalilaklinnastaṭe śuṣyati //

vairūpyaṃ sahajaṃ jarāhṛtaruciryāto yayātiḥ purā kāntyā tarjitakāmakīrtirabhavaddurdarśamūrtirnalaḥ 
saudāsasya manoharaṃ vapurabhūtsaṃtrāsanaṃ dehināṃ rūpe kasya bhaviṣyati pratidinamlāyinyanitye dhṛtiḥ //

tasmādasthirarūpaṃ vicārya rūpaṃ bhavasvarūpaṃ ca /
anurūpamadanaśamanaṃ sthirapadasaṃprāptaye sudhiyām //

pañcamo vicāraḥ

ahaṃ śūraḥ krūrapratibhaṭaghaṭāpāṭanapaṭus tarasvī senāyāṃ hayagajaghaṭānāmadhipatiḥ 
iti prauḍhaḥ puṃsāṃ nijabhujabalākrāntajagatāṃ bhavatyantardarpaḥ paribhavapadaṃ kālagalitaḥ //

śauryeṇa darpaḥ puruṣasya ko 'yaṃ dṛṣṭastiraścāmapi śūrabhāvaḥ 
aucityahīnaṃ vinayavyapetaṃ dayādaridraṃ na vadanti śauryam //

bālasya śauryaṃ kusumopamasya mātuḥ prahāre praṇayasmiteṣu 
vṛddhasya śauryaṃ śithilāṅgasaṃdheḥ svaślāghayā pūrvakathāpatheṣu //

vayastribhāge taruṇasya śauryaṃ yadeva darpaprabhavābhibhūtam 
taccittavṛttervividhasvabhāvāt paryāyaśo yātyativaiparītyam //

........... ........... 
cittasya jātyānilacañcalasya nānāguṇatvātkriyate kimasya //

hyo yena bhagnāḥ purato 'risenā bhītaḥ sa evādya bhavatyadhīraḥ 
vṛtreṇa śakraḥ samare nigīrṇaḥ phenena śakraḥ sa jaghāna vṛtram //

........ ........ 
ekaṃ samāliṅgati darpalolā kṣībeva veśyā nahi rājalakṣmīḥ //

yaḥ kārtavīryasya ca doḥsahasraṃ viccheda vīro nahi yudhi jāmadagnyaḥ /
sa sāyake rāmakarādhirūḍhe brāhmaṇyadainyapraṇayī babhūva //

rāmo 'pi sāhāyakalābhalobhāccakre kapeḥ saṃśrayadainyasevām /
śūrapratāpaḥ śiśirartuneva kālena līḍhastanutāmupaiti //

vālī prasahya plavagaḥ kareṇa sollāsakailāsasahaṃ daśāsyam 
nikṣipya kakṣāñcalasaṃdhibandhe saptābdhisaṃdhyāvidhimanvatiṣṭhat //

yuddhoddhatā bhūpatayaḥ prasiddhā baddhā jarāsaṃdhanṛpeṇa pūrvam 
sabhīmasenena bhujāyudhena dvidhā kṛtaḥ saṃdhividāraṇena //

bhīmo 'pi karṇena vikīrṇadhairyaḥ pramūḍhaśaktiḥ kṛpayā vimuktaḥ 
karṇe 'rjunasyātatakārmukasya kṣaṇātkṣaṇaṃ yācakatāṃ prayātaḥ //

tyaktvārjunaḥ kṛṣṇakalatravargaṃ jagāma gopālabalābhibhūtaḥ 
na jñāyate daivapathānuyātā śauryasya vṛttiḥ karikarṇalolā //

bhīruḥ śūratvamāyāti śūro 'pyāyāti bhīrutām 
na kvaciccapalasyāsya śauryasya niyatā sthitiḥ //

bāṇastryakṣeṇa kaṃsāricakradhārāpathātithiḥ 
ājanmabhaktipraṇayī rakṣaṇārho na rakṣitaḥ //

vegāpte kālayavane mucukundamaśiśriyat 
śauriḥ śayanaparyaṅkatalasaṃkucitākṛtiḥ //

śiśupālasya śirasi cchinne cakreṇa cakriṇā 
dṛṣṭiḥ kṛtā na cāpeṣu nṛpaistatpakṣipātibhiḥ //

bhīmaniṣpīyamāṇasṛg dṛṣṭo duryodhanānujaḥ 
aśastrābhireva strībhir droṇakarṇakṛpādibhiḥ //

sphārājagarasaṃruddhabhujadvandvo vṛkodaraḥ 
jananīkaruṇākrandaninādamukharo 'bhavat //

mahatāmapi pūrveṣām evaṃrūpā madakṣitiḥ 
sāmānyavikramoddāmaślāghā kenābhinandyate //

aśakte raudratātaikṣṇyaṃ tīvrapāpeṣu dhīratā 
chadmadhīrvāci pāruṣyaṃ nīcānāṃ śauryamīdṛśam //

niṣkāraṇanṛśaṃsasya śauryaṃ hiṃsratvamucyate 
yaḥ sarpa iva saṃnaddhaḥ prāṇabhādhāya dehinām //

etadeva paraṃ śauryaṃ yatparaprāṇarakṣaṇam 
nahi prāṇaharaḥ śūraḥ śūraḥ prāṇaprado 'rthinām //

na kaścidbuddhihīnasya śauryeṇa kriyate guṇaḥ 
parjanyagarjitāmarṣī śvabhre patati kesarī //

kiṃ śauryeṇa sarāgasya madakṣībasya dantinaḥ 
bandhakīlābhalobhena yaḥ kṣipatyavaṭe tanum //

śauryaṃ vikrītakāyasya sevakasya kimadbhutam 
meṣasyeva vadho yasya sūnābaddhasya niścitaḥ //

na darpavikṛtaṃ śauryaṃ na māyāmalinaṃ manaḥ 
na dveṣoṣṇaṃ śrutaṃ yeṣāṃ gaṇyante tadguṇā budhaiḥ //

kulaṃ kutanayeneva lobheneva guṇodayaḥ 
aiśvaryaṃ durnayeneva śauryaṃ darpeṇa naśyati //

prabhāvabhavanastambha iva dambhodbhavo 'bhavat 
saptābdhiparikhālekhamekhalāyāḥ prabhurbhuvaḥ //

tasya niḥśeṣitārāteḥ sadā yuddhamanorathaḥ 
aprāptapratimallasya yayau hṛdayaśalyatām //

sa surāsurayuddhāptadarpadarpitamānasaḥ 
kaḥ ko 'sti śūraḥ saṃrambhād ityapṛcchatsadājanam //

darpakaṇḍūladordaṇḍaṃ pṛcchantaṃ rabhasena tam 
sarvāvamānasaṃnaddhaṃ jagādābhyetya nāradaḥ //

nāsti tvatsadṛśaḥ śūras trailokye satyamucyate 
kiṃ tu jāne raṇārhau te naranārāyaṇāvṛṣī //

badaryāśramasaṃsaktau tīvre tapasi niṣṭhitau 
yuddhecchau tau yadi syātāṃ tatpūrṇaste manorathaḥ //

nāradenetyabhihite sa badaryāśramaṃ yayau 
vilokayannijabhujau pratyāsannaraṇotsavau //

dṛṣṭvā tejonidhī tatra naranārāyaṇau nṛpaḥ 
manorathapathābhyastaṃ yayāce yuddhamuddhataḥ //

taṃ yuddhakāmukaṃ tiryagdṛśā gambhīradhīrayā 
vilokyovāca sāvajñasmitadigdhādharaṃ naraḥ //

mahīpate nivartasva na vayaṃ yuddhakovidāḥ 
yuktastaireva saṅgrāmas tava ye bhūmyanantarāḥ //

ityukto 'pi yadā rājā na cacāla raṇādarāt 
tadā taṃ dṛptamaiṣīkaniśitāstrairapūrayat //

pradīptajvalanākāraiḥ śarairākīrṇavigrahaḥ 
vinaṣṭavigraharucir nṛpastatyāja dhīratām //

akāṇḍakhaṇḍitoccaṇḍadarpajvarabharo nṛpaḥ 
kṛpaṇaḥ prāṇarakṣāyai tameva śaraṇaṃ yayau //

vāritāstrastatastena bhagnamānamanorathaḥ 
lajjāvikuṇṭhakaṇṭhaḥ svāṃ rājadhānīṃ yayau nṛpaḥ //

iti mānasya mahatām api ghorāśanirmadaḥ 
lohasya svamaleneva kṣayo darpeṇa tejasaḥ //

tasmātsadā mānadhanena puṃsā darpaḥ prayatnena nivāraṇīyaḥ 
darpogravaktrasya suhṛjjano 'pi sarvātmanā tivranipātasajjaḥ //

adarpaśauryaspṛhaṇīyasattvā goviprarakṣākṣapitasvadehāḥ 
prayānti vīrāḥ sukṛtāmṛtādrair yaśaḥśarīrairajarāmaratvam //

ṣaṣṭho vicāraḥ

jagatyeko bhadradvirada iva dānārdrasaraṇir yaśasvī niḥsvānāmahamabhimatāśāphalataruḥ 
iti tyāgodagraṃ vahati kila darpaṃ manasi yas tadudbhūtaṃ sarvaṃ sukṛtamapahāya vrajati saḥ //

svargādisaṃbhogaphalābhilāṣāt pātrāya pūjāṃ pratipadyate yaḥ 
dharmārthapaṇyakrayavikrayo 'sau kastena dānaprabhavo 'bhimānaḥ //

yadvidyādiguṇotkarṣaviśeṣaparitoṣitaiḥ 
dīyate prītidhanayoḥ sa paṇyakrayavikrayaḥ //

lokaprasiddhisiddhyai yaḥ prayacchati guṇastavaiḥ 
karoti vittayaśasoḥ sa tadā krayavikrayam //

avamānahataṃ yacca dattamaśraddhayā dhanam 
ūṣare niṣphalaṃ bījaṃ kṣiptamakṣiptameva tat //

tyāgino 'nyasya saṃgharṣe kīrtyutkarṣajigīṣayā 
dattaṃ kāraṇabhūtasya tasyaivānte phalapradam //

parārtiśamanaṃ vittam ajñātamanudīritam 
aphalākāṅkṣayā yuktaṃ prayātyalpamanalpatām //

kurukṣetrādideśeṣu kāleṣvarkagrahādiṣu 
ātmopakāramātreṇa pātre dānena kiṃ madaḥ //

deśakālakriyāpātrāṇy avicāryaiva kevalam 
pareṣāmārtiśamanaṃ dayārdraṃ dānamucyate //

rakṣāyai saṃpadāṃ putrakalatrasukhasiddhaye 
dīyate yatprayatnena lobhadānena tena kim //

vāde khalaiḥ khalīkṛtya vedhadrutaparīkṣayā 
dīyate yaccirakliṣṭaṃ kaṣṭadānena tena kim //

tyaktvāśāgatasatpātraṃ pūrṇāyābhyarthya dīyate 
yattaducchvāsasaṃtaptaṃ dagdhadānena tena kim //

anyadābhāṣitaṃ pūrvaṃ dattamanyattato 'lpakam 
yatsadoṣamayogyaṃ vā kūṭadānena tena kim //

cirasevānurodhena lobhakṛcchrādanicchayā 
aprasādena yaddattaṃ balādānena tena kim //

yatpuṣpadhūpatilakapratipattipradarśitam 
dattamatyalpaniḥsāraṃ dambhadānena tena kim //

prabhūtabhārasaṃbhāraṃ rājacaurādiviplave 
dattvā yadduṣṭamudghuṣṭaṃ śalyadānena tena kim //

anāsvādyamavikreyam anādeyamanīpsitam 
dattaṃ nirupakāraṃ yad vandhyadānena tena kim //

ṛṇavaccirasaṃśodhyaṃ vacasā pratipāditam 
yannityayācanadveṣaṃ yācyadānena tena kim //

ekasmai pūrṇamanyasmai kṛśaṃ tulyaguṇodaye 
bhedādyadarpitaṃ rāgadveṣadānena tena kim //

ṛṇadaiḥ svajanaiḥ putrair labdhakṣāmapratigrahaḥ 
nityamāyāsyate yena kalidānena tena kim //

na parasyārtiśanamaṃ nātmanaḥ puṇyakāraṇam 
dattālpamūlyenāptaṃ yat svalpadānena tena kim //

durgraheṣu viruddheṣu daśāpāke 'tidāruṇe 
dīyate doṣaśāntyai yad bhayadānena tena kim //

mumūrṣastyaktasarvāśaḥ śayanastho dadāti yat 
mūrcchāsthānena manasā mohadānena tena kim //

dattaṃ priyaviyogograśokaśalyārtacetasā 
yatpaścāttāpajananaṃ bāṣpadānena tena kim //

purohitāya gurave śāntisvastividhāyine 
dīyate yatprasaṅgena bhṛtidānena tena kim //

yatsaṃtyaktaphalaspṛhaṃ yaducitaṃ sarvasvabhūtaṃ ca yan nānyāyena yadarjitaṃ paradhanasparśena śaptaṃ na yat 
dattvā duḥkhaśataṃ na yatsvavacasā paścānna yadgaṇyate taddānaṃ dhanabījavāpanipuṇaḥ śeṣaḥ prakāraḥ kṛṣeḥ //

prāptuṃ svargavarāṅganāstanataṭasparśātiriktaṃ sukhaṃ datto merurapi prayāti tṛṇatāmātmopakārecchayā 
āpannārtivilokane karuṇayā śraddhāsudhāpūritaṃ sattvotsāhasamanvitaṃ tṛṇamapi trailokyadānādhikam //

yudhiṣṭhirasya bhūbhartuḥ purā kanakavarṣiṇaḥ 
aśvamedhe vidhānena vartamāne mahākratau //

sajjāsu rājabhojyāsu vividhāsvannapāliṣu 
aniśaṃ ratnapātreṣu bhuñjāneṣu dvijanmasu //

vipreṣu pūryamāṇeṣu maṇikāñcanaśāsanaiḥ 
ucchiṣṭabhūmiṃ nakulaḥ svabilātsamupāyayau //

dīptakāñcanavarṇena pārśvenaikena śobhitaḥ 
apareṇāsuvarṇena vitīrṇajanakautukaḥ //

so 'bhyetya tūrṇamucchiṣṭa hemapātracyute 'mbhasi 
luloṭha śapharotphālaparivartavivartanaiḥ //

suvarṇapārśvaṃ nakulaṃ dṛṣṭvā sarve kutūhalāt 
majjantamucchiṣṭajale kṣitipāya nyavedayan //

prāptena bhūbhujā dṛṣṭvā dṛṣṭyā pṛṣṭa iveṣṭayā 
so 'vadadvismayabhuvā suspaṣṭākṣarayā girā //

rājannasyātidānasya na paśyāmyucitaṃ phalam 
asmātprabhūtasaṃbhārāt saktupātraṃ varaṃ param //

pravṛtte 'sminmahādāne mahataste mahīpate 
vitte vṛtte ca citte ca śuddhiṃ ko vetti tattvataḥ //

saṃbhāro 'yaṃ bhuvanabhavanavyāptiparyāptabhogaḥ sarvāśāsu pratatajanatāpūraṇe 'tyantatucchaḥ 
āpannārtipraśamanavidhau sattvaśuddhipradāne saṃnaddhānāmapi tṛṇakaṇaḥ kāñcanādritvameti //

śrūyatām yanmayā dṛṣṭaṃ bhūpate svayamadbhutam 
udetyudīrite yasmin kāye romāñcakañcukaḥ //

śiloñchavṛttinā pūrvaṃ vipreṇa kṣetracāriṇā 
upavāsakṛśenāptaṃ yavastokaṃ kalatriṇā //

saktupātre tataḥ siddhe kṛtadevapitṛkriyaḥ 
jāyāputravibhāgena svaṃ bhāgaṃ bhoktumudyayau //

sa prāṇāhutitoyārthī dadarśātithimāgatam 
kṣutkṣāmakukṣiṃ saṃkṣiptasarvāṅgaśithilākṛtim //

tasmai vihitasatkāraḥ saprasādena cetasā 
śraddhāsudhāvasiktaṃ tat sa dadau nijabhojanam //

nigīrṇo 'tithinā tasminn akṣīṇakṣudvikāriṇā 
tadbhāryāpyādaravatī tasmai svamaśanaṃ dadau //

tenāpyatṛptimālokya tatsūnuḥ śraddhayātithim 
svabhojanena vidadhe saṃpūrṇāśananirvṛttam //

gate bhuktvātithau tasminn upavāsakṛśo dvijaḥ 
sattvotsāhayutastasthau klānto 'pi niśi nirvyathaḥ //

athāhaṃ saktugandhena nirgataḥ kṣudhito bilāt 
prāptastatparṇakuṭikām utsṛṣṭocchiṣṭavartinīm //

tatrācamanatoyena spṛṣṭamātrasya me nṛpa 
pasya me dakṣiṇaṃ pārśvaṃ jataṃ hemamayacchavi //

tato 'haṃ vāmapārśvasya hemacchāyāptaye sadā 
nirnidraścintayā yātaḥ kṛśatāmeva kevalam //

yadyatprāpnoti puruṣaḥ karmayogātsamīhitam 
tattatsaṃpūraṇāyaiva yāti cintāvidheyatām //

adhunā vartamāne 'sminn aśvamedhe tava kratau 
hemapārśvāśayāyāto viprocchiṣṭāmahīmaham //

ratnakāñcanapātrāmbusiktasya luṭhataściram 
mama kāntilavo 'pyaṅge na kaścidiha dṛśyate //

sarvathā sattvaśuddhāya dānāyātilaghīyase 
namo mahāphalāyaiva na bhogāṅgaprasaṅgine //

ityuktvā nakule yāte tattatheti yudhiṣṭhiraḥ 
vicintya saṃtatocchvāsaḥ kṣaṇaṃ stimitatāṃ yayau //

tasmātsuvarṇāmbararatnabhūmidānairna darpaḥ puruṣeṇa kāryaḥ 
bhavatyudāraṃ karuṇārdrasattvaṃ dānaṃ sadā kasyacideva puṇyaiḥ //

saptamo vicāraḥ

tapaḥ sadā rāgadhanābhimānamohaprahāṇāya satāmabhīṣṭam 
tenaiva darpo yadi kiṃ vṛthaiva tyakto nikāyaḥ kṣapitaśca kāyaḥ //

sarvātmanā śuddhadhiyā vidheyaḥ saṃsāradoṣapraśamāya yatnaḥ 
kopopataptaṃ dhanarāgadigdhaṃ karoti tīvraṃ na tapaḥ praśāntim //

cittaṃ viraktaṃ yadi kiṃ tapobhiś cittaṃ sarāgaṃ yadi kiṃ tapobhiḥ 
cittaṃ prasannaṃ yadi kiṃ tapobhiś cittaṃ sakopaṃ yadi kiṃ tapobhiḥ //

kopena śāpaśpuritādharāṇāṃ kāmena kampasphuritādharāṇām 
svedāmbhasā tulyasamudbhavena nistejasāṃ kiṃ tapasā munīnām //

bhāryāpyahalyā kila gautamasya kruddhasya śāpena śilā babhūva 
nīto vasiṣṭhena ruṣābhiśaptaś caṇḍālatāṃ bhūmipatistraśaṅkuḥ //

bhūmagnamūrtirvararatnalobhād vipāṭayantīṃ nayane sukanyām 
tatpāṇisaṃsparśasukhādareṇa sehe nikāraṃ cyavanaḥ sarāgaḥ //

pāṇḍuḥ priyākaṇṭhavilambibāhur yayau stananyastatanuryadastam 
dagdhaḥ parīkṣitphaṇiphūtkṛtairyat tapasvikopasya vijṛmbhitaṃ tat //

visārasaṃsāratarorna yena niḥśeṣamunmūlitameva mūlam 
śāpopatāpaprabhuṇā pareṣāṃ kiṃ tena mithyātapasā munīnām //

na rājasevārajasā viluptaṃ na bhūmividyādivivādataptam 
na dambhadīkṣākuhakākulaṃ yat kalyāṇamittraṃ vimalaṃ vrataṃ tat //

sasaṃcayaṃ guptakalatraputraṃ punargṛhītavyavāharabhāram 
dambhābhimānodbhavakaṣṭabhūtaṃ mithyāvrataṃ jīvitavṛttyupāyaḥ //

sarāgarogaṃ bahulapramohaṃ sarogasaṃbhāramakharvagarvam 
pradveṣadoṣoṣṇamameyamāyaṃ saṃsāracihnaṃ vratametadagryam //

jaṭākṣasūtrājinayogapaṭṭakanyādṛḍhagranthinipīḍyamānam 
vivekahīnaṃ virataprakāśaṃ vrataṃ bṛhadbandhanamāmananti //

sarāgakāṣāyakaṣāyacittaṃ śīlāṃśukatyāgadigambaraṃ vā 
laulyodbhavadbhasmabharaprahāsaṃ vrataṃ na veṣodbhaṭatulyavṛttam //

niḥsaṅgayogaṃ dhanabhogasaṅgaṃ vilambikaṅkālakapālamālam 
kopākulaṃ sparśavivarjanīyaṃ bhāravrataṃ tatkathayanti pāpam //

bālastapasvī kimato 'sti hāsyaṃ yuvā vanaiṣī kimato 'styayogyam 
vṛddha sarāgaḥ kimato 'sti nindyaṃ mūrkhaḥ pramātā kimato 'sti śocyam //

kṣamā śamaḥ śāsanamindriyāṇāṃ manaḥ prasiktaṃ karuṇāmṛtena 
tapo 'rhametatsajane vane vā kāyasya saṃśoṣaṇamanyadāhuḥ //

himācale śyāmaladevadāruvane purā nirjharacāruhāsye 
tapasyatāṃ śoṣajuṣāṃ munīnāṃ kālo yayau varṣasahasrasaṃkhyaḥ //

tataḥ kadācidbhagavānbhavārtihārī vihārāya nabhaḥpathena 
samaṃ bhavānyā vṛṣabhādhirūḍhaḥ samāyayau śītamayūkhamauliḥ //

tasyoditānāṃ vadanaprabhāṇāṃ dīrghīkṛtānekaśaśiprabhāṇām 
vilāsahāsyena nabho babhūva vibhaktisaṃsaktasitottarīyam //

devī vilokyātha tapaḥprayatnatīvraprayāsaprakaṭāsthiśeṣān 
munīnkṛpāveśaviṣaṇṇacittā śaśāṅkalekhābharaṇaṃ babhāṣe //

deva tvadārādhananiścalānāṃ saṃtyaktasarvāgrahanigrahāṇām 
tapa kriyāśoṣitavigrahāṇāṃ nādyāpi muktiḥ kimaho munīnām //

kasmādamī varṣasahasralagnakleśāvalagnāstanuśoṣamagnāḥ 
bhavatpadaṃ nityasukhāya naiva nirāmayaṃ tanmunayaḥ prayānti //

pṛthuḥ prasādaḥ prathamāgateṣu nirādaratvaṃ cirasaṃśriteṣu 
svācchandyalīlāvipulāvalepād eṣu svabhāvaḥ sulabhaḥ prabhūṇām //

iti priyāyāḥ praṇayopapannam ākarṇya vākyaṃ giriśo 'bravīttām 
kurvanviṣaśyāmalakaṇṭhakāntiṃ dantaprabhābhiḥ pratibhāvihīnam //

devi tvayoktam dayayā munīnāṃ bhaktānurodhāducitaṃ mamaitat 
eṣāṃ bhavollaṅganavighnabhūtau śāntiṃ gatau kiṃ tu na kāmakopau //

vanapraveśairniyamairaśeṣaiḥ kriyāviśeṣaiḥ kṛtakāyaśoṣaiḥ 
na nirvikāraṃ padamāpnuvanti kopena kāmena ca kṛṣyamāṇāḥ //

pratyakṣameṣāṃ manaso vikāraṃ saṃdarśayāmyeṣa niṣaktamantaḥ 
tīvravrataiḥ śuṣyati kāya eva na vāsanālīnaghanapramohaḥ //

saṃtyaktabhogāḥ spṛhayā vimuktāḥ snehe 'pyarāgāḥ sujane 'pyasaṅgāḥ 
bhajantyavikleśatapaḥprasaktā yuktāḥ prakāmaṃ padamavyayaṃ tat //

uktveti śaṃbhurvṛṣabhātsalīlaṃ girerivāgrādavatīrya bhūmim 
kṣaṇādabhūdadbhutarūparāśir nagnavrataḥ kāntisudhāvadātaḥ //

tasyāmarādhīśakirīṭaratnaśoṇaprabhārdrāviva pādapadmau 
pracakraturvidrumabālavallīnavaprarohādbhutagarvamurvyām //

suspaṣṭajānu pracitoruśobhi nābhihradāvartavibhaktamadhyam 
tattasya rūpaṃ pravilambibāhoḥ pīnāṃsamāsīnmukhapūrṇacandram //

ananyalāvaṇyasudhābdhimadhyasnātairivāṅgaiḥ sphaṭikāvadātaiḥ 
cakre daśāśāḥ sa pṛthuprakāśā digambaratvādiva jātahāsāḥ //

pāṇisthitaśyāmamayūrapicchacchāyācchaṭāvicchurito 'sya kaṇṭhaḥ /
rarāja līnāntarakālakūṭamiṣāgninevārpitadhūmalekhaḥ //

sa locanābhyāṃ pṛthupakṣmalābhyām āraktaparyantamanoharābhyām 
vyadhādivānaṅganavāṅgasaṅge digaṅganānāmanurāgadīkṣām //

dṛṣṭvā trilokīkalitābhilāṣaṃ vapuḥ smarārerjanitasmaraṃ tat 
pūrvāpakārasmṛtijātalajjaṃ cakṣuḥ kṣaṇaṃ kvāpi yayau tṛtīyam //

babhau sa kāntaḥ kuṭilāsitena skandhaspṛśā kuntalasaṃcayena 
anveṣṭumiṣṭāṃ mukuṭendulekhāṃ niśāgaṇeneva samāgatena //

latāvadhūpallavapāṇimuktaiḥ smitāvadātairvibabhau ca puṣpaiḥ 
rūpāntare nihnutajahnukanyāphenāvaśeṣairiva kīrṇakeśaḥ //

tenānyarūpeṇa kṛtā navaiva kāntiścakāśe nijarūpaguptyai 
jūṭādivendurmṛditaḥ karābhyāṃ sarvāṅgamabhyaṅgapade niyuktaḥ //

rūpaṃ virūpīkṛtamanmathasya tattasya kāntyā kamanīyamāsīt 
lajjāpahārādvanadevatānāṃ savismayo yena navābhilāṣaḥ //

vivāsasastasya sasaṅgamaṅge lajjāvatīnāṃ spṛhayaiva petuḥ 
netrāṇi vidyādharasundarīṇāṃ līlāravindārdhatiraskṛtāni //

nabhaḥsthitānāṃ tridaśāṅganānāṃ tadgātrasaundaryavaśīkṛtānām 
prakampaśiñjānavibhūṣaṇānāṃ netrotsavo 'bhūdgativighnabhūtaḥ //

susiddhakanyāñjalipallavāgravimuktanīlotpalapuṣpakuñjam 
aṅge jagallocanavargamasya saundaryasaṃsaktamivābabhāse //

taddarśane kautukaniścalānāṃ karṇāvataṃsīkṛtalocanānām 
mṛgāṅganānāmapi saspṛhābhūn nitāntamantaḥkaraṇapravṛttiḥ //

tasya praveśe vadanādhivāsalobhabhramadbhṛṅgagaṇāñcitānām 
abhūtsajṛmbhaśvasanākulānāṃ muhurlatānāṃ kusumeṣu kampaḥ //

śanaiḥ śanairāśramasaṃnikarṣaṃ taṃ yauvanaṃ mūrtamivāpatantam 
vilokya kāntaṃ munikāminīnāṃ manaḥ praharṣocchalitaṃ babhūva //

tāsāṃ tadālokananirnimeṣā dṛṣṭiḥ paraṃ karṇapathapraviṣṭā /
utsṛṣṭalajjāvipulābhilāṣādasūcayanmugdhamṛgīvilāsam //

tāsāṃ tadarcārabhasotthitānāṃ srastāṃśukotkampighanastanīnām 
navena kāmena khalīkṛtānāṃ jṛmbhābhavo 'bhūdbhujayorvilāsaḥ //

tāsāṃ babhau romalatā mukhendubhītā tamaḥśrīḥ stanarakṣiteva 
rāgāgnidhūmaprasarāgryalekhā tanīyasī nābhivinirgateva //

tasyādhare cumbanalālaseva kaṇṭhe haṭhāliṅganasaspṛheva 
hṛdi stananyāsasamutsukeva papāta dṛṣṭiḥ sahasaiva tāsām //

bhṛṅgasvanairāhitahuṃkṛtābhiḥ puṣpyatprasūnaiḥ prasṛtasmitābhiḥ 
vātāñcitaiḥ pallavapaṇibhistā nivāryamāṇā iva mañjarībhiḥ //

saṃtarjyamānā iva homadhūmalekhāvalībhrūbhramaṇena digbhiḥ 
tasyāntike śīladukūlamuktisajjā vilajjāḥ prasabhaṃ babhūvuḥ //

niḥśvāsinīnāṃ smarabāṇapuṅkhapakṣāntavātairiva kampitānām 
tāsāṃ vilokyaiva manovikāraṃ bhrūbhaṅgabhīmā muniparṣadāsīt //

kopotkaṭavyāghravidīryamāṇakṣamāmṛgīraktaciteva teṣām 
āsannadoṣāgamavāsarāntasaṃdhyānibhābhūtsahasaiva dṛṣṭiḥ //

daṣṭādharāḥ kampavidhūrṇamānāḥ svedārdradehā viṣamaṃ śvasantaḥ 
te bhejire rāgasamudgaterṣyāḥ kopākulāḥ kāmukavṛttameva //

antarjvalatkopakṛśānudhūmasaṃkāśakṛṣṇājinabaddhakakṣaḥ 
tridaṇḍamudyamya javena kaścid abhyādravannagnatanuṃ vṛṣāṅkam //

bṛsīṃ samutkṣipya sakampabāhuś cikṣepa kaścitkṣamayā vihīnaḥ 
yenāsanātkṣmāvirahādivāśu mohe nirālambatanuḥ papāta //

kamaṇḍaluṃ kaścidakāṇḍacaṇḍasaṃrambhapiṇḍīkṛtakopatulyam 
ādāya mohena pinākapāṇeḥ puraḥ prahārābhimukho babhūva //

teṣāmamarṣādbhṛśamakṣamāṇāṃ soḍhuṃ nikāraṃ kṣaṇamakṣamāṇām 
prāpuḥ prayātāḥ kṣitimakṣamālā bhrūbhaṅgatāṃ tasya tapovanasya //

tatsaṃbhramādāśramamañjarīṇāṃ kampākulānāṃ kusumāntarotthaiḥ 
āsītpramohapratimo 'ndhakāraḥ śāpākṣarāmairbhramarairbhramadbhiḥ //

te prāpurīrṣyāpadamandhakāri vaktraṃ śaśāṅkopamamīkṣamāṇāḥ 
kaṇṭhasthalālokanakālakūṭasaṃpūritākṣā iva mohamūrcchām //

te taṃ smitaprasphuritādharāgram udagralāvaṇyaviśeṣatarṣāḥ 
patnīvikārogranikāramūcuḥ kaṇṭhāntaraśvāsavikīrṇavarṇāḥ //

ko 'yaṃ vijātirviguṇaḥ kalāvān nagno vṛṣāṅkaḥ praviśatyalajjaḥ 
pradūṣitā yena maharṣijuṣṭā gaṅgeva śuddhā lalanāvalīyam //

anena saṃsūcayatā nigūḍharūpeṇa darpādakulīnabhāvam 
nītā pavitratvamiyaṃ munīnāṃ kāpālikeneva vanāntabhūmiḥ //

aho batāsya pratibhā prasahya satīsamāliṅganasaspṛhasya 
kenāpi kāmatkuhakakrameṇa kāntaṃ kṛtaṃ rūpamanena nūnam //

uktveti tasmai sasṛjuḥ sakopās te daṇḍapāṣāṇabṛsīśatāni 
dveṣāvṛtākṣṇāmavivekajanmā mohaḥ pramāde gurutāmupaiti //

dūre bhavatyatha śanaiḥ śiśirāṃśumaulau teṣaṃ prakopavipulānalatāpitānām 
taddarśanānusaraṇaprasṛtasya yatnaḥ patnījanasya sutarāṃ vinivartane 'bhūt //

atha sa bhagavānbhargaḥ svargāpagāpṛthunirjharaprasṛtahasitastasmādeśātkrameṇa tirohitaḥ 
praśamavimalaṃ vyoma vyāpya priyāmavadatsmayasmitasitamukhīṃ dṛṣṭaṃ devi tvayā municeṣṭitam //

bhasmasmeraśarīratā pṛthujaṭābandhaḥ śiromuṇḍanaṃ kuṇḍī daṇḍakamaṇḍalupraṇayitā carmākṣasūtragrahaḥ 
kāṣāyavyasanaṃ nirambararuciḥ kaṅkālamālādhṛtiḥ kāmakrodhavaśādviśeṣaruciraṃ sarvaṃ vṛthaiva vratam //

darpotkopātparaṇitajaṭāsūtrabandhacca mohād antaḥsīdatsarasaviṣayāsvādasaṃvādasaṅgāt 
āśāpāśavyasananicayādvāsanālīnadoṣān naiṣāṃ muktirbhavati tapasā kāyasaṃśoṣaṇena //

ityuktaṃ tripurāriṇā girisutā śrutvā yathārthaṃ vaco niścitya vratamapraśāntamanasāṃ mithyaiva kāyakṣayam 
saṃsāroparamāya moharajasaḥ śāntyai munīnāṃ paraṃ rāgadveṣavimuktaye ca dayayā cakre harasyārthanām //

devyārthito 'tha bhagavānkṛpayā smarāris teṣāmanugrahamayena vilokanena 
cakre smitasnapitadigvadano munīnāṃ līnasya moharajasaḥ sahasaiva śāntim //

tapoviśeṣairniśitaprayatnais tasmānna kāryaḥ pṛthumohadarpaḥ 
dveṣeṇa rāgeṇa mohadayena tapaḥ kṣayaṃ yāti saha smayena //

praśānto 'ntastṛṣṇāviṣamaparitāpaḥ śamajalair aśeṣaḥ saṃtoṣāmṛtavisarapānena vapuṣaḥ 
asaṅgaḥ saṃbhogaḥ kamaladalakīlālatulyā bhavāraṇye puṃsāṃ parahitamudāraṃ khalu tapaḥ //
